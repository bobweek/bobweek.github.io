Lawrence C. Evans

Second-Order Elliptic Equations

6.1 Definitions

We study problems with Dirichlet boundary conditions:

\[\begin{equation} \left\{\begin{matrix} Lu = f & x\in U \\ u = 0 & x\in \partial U \end{matrix}\right. . \end{equation}\]

where \(L\) is a second order partial differential operator appearing as

  • divergence form (best used with energy methods):

\[\begin{equation} Lu = -\sum_{i,j}\left(a^{ij}(x)u_{x_i}\right)_{x_j}+\sum_ib^i(x)u_{x_i}+c(x)u \end{equation}\]

  • nondivergence form (best used with maximum principle techniques):

\[\begin{equation} Lu = -\sum_{i,j}a^{ij}(x)u_{x_i x_j}+\sum_ib^i(x)u_{x_i}+c(x)u \end{equation}\]

Def: \(L\) is uniformly elliptic if there exists a constant \(\theta>0\) s.t.

\[\begin{equation} \sum_{i,j}a^{ij}(x)\xi_i\xi_j\geq\theta|\xi|^2 \end{equation}\]

for a.e. \(x\in U\) and for all \(\xi\in\mathbb{R}^n\). \(\diamond\)

  • If \(L\) is elliptic, then for each \(x\in U\) the matrix \(\pmb{A}(x)=(a^{ij}(x))\) is positive definite, with smallest eigenvalue \(\geq0\).

  • If \(a^{ij}=\delta_{ij}, \ b^i=0, \ c=0\), then \(L=-\Delta\)

  • Population dynamics: If we set \(f=u_t\), the terms \(a\) represent diffusive migration, \(b\) directional migration/taxis, \(c\) the Malthusian fitness. However, this is not technically an elliptic equation…

6.1.2 Weak solutions

Assume \(a^{ij},b^i,c\in L^\infty(U)\) & \(f\in L^2(U)\) and that \(L\) is in divergence form \((2)\).

Def:

  1. The bilinear form \(B[ \ , \ ]\) associated with the divergence form elliptic operator \(L\) is

\[B[u,v]\equiv\int_U\left(\sum_{ij}a^{ij}u_{x_i}v_{x_j}+\sum_ib^iu_{x_i}v+cuv\right)dx\]

for \(u,v\in H_0^1(U)\).

  1. \(u\in H_0^1(U)\) is a weak solution of the BVP \((1)\) if

\[B[u,v]=(f,v)\equiv\int_U f(x)v(x)dx\]

for all \(v\in H_0^1(U)\).

Characterization of \(H^{-1}\)

  1. For \(f\in H^{-1}\) there exist functions \(f^0,\dots,f^n\in L^2\) (\(n=\dim(U)\)) s.t.

\[\left<f,v\right>=\int_U\left(f^0v+\sum_if^iv_{x_i}\right)dx, \ (v\in H_0^1)\]

  1. Also,

\[\|f\|_{H^{-1}}=\inf\left\{\left(\int_U\sum_i|f^i|^2dx\right)^{1/2}: f\text{ satisfies i. for the specified }f^0,\dots,f^n\in L^2\right\}\]

  1. In particular,

\[(v,u)_{L^2}=\left<v,u\right>\]

for all \(u\in H^1_0, \ v\in L^2\subset H^{-1}\).

Notation: Write “\(f=f^0-\sum_if^i_{x_i}\)” whenever i. holds.

Consider the boundary value problem

\[\begin{equation} \left\{ \begin{matrix} Lu = f^0-\sum_i f^i_{x_i} & x\in U \\ u = 0 & x\in\partial U \end{matrix} \right. \end{equation}\]

with \(L\) in divergence form and \(f^i\in L^2(U)\). From above we see \(f=f^0-\sum_if^i_{x_i}\in H^{-1}(U)\).

Def: \(u\in H_0^1(U)\) is a weak solution of (5) if

\[B[u,v]=\left<f,v\right>\]

for every \(v\in H_0^1(U)\), where \(\left<,vf\right>\equiv\int_Uf^0v+\sum_if^iv_{x_i}dx\) and \(\left< \ , \ \right>\) is a “pairing” (like an innder-product) between \(H^{-1}\) and \(H_0^1\).

  • If a BVP has condition \(u=g\) on \(\partial U\), so long as \(g\) is the trace of some \(w\in H^1\) so that \(u-w\in H_0^1\), the BVP can be rephrased with a Dirichlet condition be subtracting \(g\) from both equations.

  • Because of this we focus on Dirichlet boundary conditions.

6.2 Existence of weak solutions

Lax-Milgram Theorem: If \(B:H\times H\rightarrow\mathbb{R}\) is a bilinear form for which there exists \(\alpha,\beta>0\) s.t.

  1. \(|B[u,v]|\leq\alpha\|u\| \ \|v\|, \ \ \ (u,v\in H)\)

  2. \(\beta\|u\|^2\leq B[u,u], \ \ \ (u\in H)\)

and if \(f:H\rightarrow\mathbb{R}\) is a bounded linear functional on \(H\), then there exists a unique element \(u\in H\) s.t. \(B[u,v]=\left<f,v\right>\) for all \(v\in H\). \(\diamond\)

  • This theorem is just Riesz representation if \(B\) is symmetric. Hence, it its significance is that it works for asymmetric \(B\).

6.2.2 Energy estimates (applying Lax-Milgram)

Assume \(B\) is the bilinear form associated with \(L\).

Theorem 2 (Energy estimates): There exists \(\alpha,\beta>0\) and \(\gamma\geq0\) s.t.

  1. \(|B[u,v]|\leq\alpha\|u\|_{H_0^1} \ \|v\|_{H_0^1}\)

  2. \(\beta\|u\|_{H_0^1}^2\leq B[u,u]+\gamma\|u\|^2_{L^2}\)

Proof: Checking the first condition is just a straightforward use of \(|\int f|\leq\int|f|\). The second requires the ellipticity condition \((4)\). \(\square\)

  • When \(\gamma>0\), \(B\) does not satisfy hypothesis of Lax-Milgram. So we need the following:

Theorem 3 (first existence theorem): There is a number \(\gamma\geq0\) s.t. for all \(\mu\geq\gamma\) and each \(f\in L^2(U)\), there is a unique weak solution \(u\in H_0^1\) of the BVP

\[\begin{equation} \left\{ \begin{matrix} Lu+\mu u=f & x\in U \\ u = 0 & x\in\partial U. \end{matrix} \right. \end{equation}\]

Proof: Set \(L_\mu=Lu+\mu u\). Then the associated \(B_\mu\) satisfies Lax-Milgram. With \(f\) fixed in \(L^2\), setting \(\left<f,v\right>=(f,v)\) defines a bounded linear functional on \(L^2\). Apply Lax-Milgram to find unique \(u\in H_0^1\) satisfying \(B_\mu[u,v]=\left<f,v\right>\) for all \(v\in H_0^1\). \(\square\)

\(H_0^1\sim H^{-1}\):

  • For all \(f^i\in L^2\), \((i=0,\dots,n)\), \(\left<f,v\right>=\int_U f^0v+\sum_i f^iv_{x_i}\) is a bounded linear functional on \(H_0^1\). Hence, there is a weak solution \(u\) of the PDE

\[\begin{equation} \left\{ \begin{matrix} Lu+\mu u=f^0-\sum_if^i_{x_i} & x\in U \\ u = 0 & x\in\partial U. \end{matrix} \right. \end{equation}\]

  • In particular, \(L_\mu=L+\mu I:H_0^1\rightarrow H^{-1}\) is an isomorphism.

6.2.3. Fredholm alternative

We first collect some results from the appendix.

  • \(X,Y\) are Banach spaces

Def: An operator \(A:X\rightarrow Y\) is closed if whenever \(u_k\rightarrow u\) and \(Au_k\rightarrow v\), then \(Au=v\).

Theorem (closed graph theorem): If \(A\) is a closed linear operator, then it is bounded.

Def: \(A:X\rightarrow X\) is a bounded operator.

  1. The resolvent of \(A\) is \(\rho(A)=\{\eta\in\mathbb{R}:(A-\eta I)\text{ is a bijection }\}\).

  2. The spectrum of \(A\) is \(\sigma(A)=\mathbb{R}-\rho(A)\).

  3. (eigenvalues) The point spectrum of \(A\) is \(\sigma_p(A)=\{\eta\in\sigma:N(A-\eta I)\neq\{0\}\}\)

\[\diamond\]

  • If \(\eta\in\rho\), the closed graph theorem implies that \((A-\eta I)^{-1}:X\rightarrow X\) is a bounded linear operator.

Def: Weak convergence is denoted by \(u_k\rightharpoonup u\) which implies \(\left<u^*,u_k\right>\rightarrow\left<u^*,u\right>\) for every bounded linear functional \(u^*\in X^*\).

Def: A bounded linear operator \(K:X\rightarrow Y\) is compact if for every bounded sequence \(\{u_k\}\subset X\), the sequence \(\{Ku_k\}\) is precompact in \(Y\) (which occurs when there exists a subsequence \(\{u_{k_j}\}\) s.t. \(\{Ku_{k_j}\}\) converges in \(Y\)).

  • If a linear operator \(K:H\rightarrow H\) is compact and \(u_k\rightharpoonup u\), \(Ku_k\rightarrow Ku\).

Def: The adjoint of an operator \(K\) is defined by \(\left<Ku,v\right>=\left<f,K^* v\right>\).

Theorem: If \(K\) is compact, so is \(K^*\).

Theorem (Fredholm alternative: \(K:H\rightarrow H\) a compact operator. Then

  1. \(N(I-K)\) is finite dimensional
  2. \(R(I-K)\) is closed
  3. \(R(I-K)=N(I-K^*)^\bot\)
  4. \(N(I-K)=\{0\}\) iff \(R(I-K)=H\)
  5. \(\dim N(I-K)=\dim N(I-K^*)\)

\[\diamond\]

The Fredholm alternative implies precisely one of the two statements:

  1. For each \(f\in H\), the equation \(u-Ku=f\) has a unique solution.

  2. The homogeneous equation \(u-Ku=0\) has solutions \(u\neq0\). The space of these solutions is finite-dimensional and the nonhomogeneous equation \(u-Ku=f\) has a solution iff \(f\in N(I-K^*)^\bot\).

Theorem: If \(\dim H=\infty\) and \(K\) is compact, then

  1. \(0\in\sigma(K)\)
  2. \(\sigma(K)-\{0\}=\sigma_p(K)-\{0\}\)
  3. \(\sigma(K)-\{0\}\) is finite or else is a sequnce tending to \(0\).

Def: For linear bounded symmetric \(S:H\rightarrow H\) denote

\[m\equiv\inf_{\begin{matrix} u\in H \\ \|u\|=1 \end{matrix}}(Su,u), \ \ \ M\equiv\sup_{\begin{matrix} u\in H \\ \|u\|=1 \end{matrix}}(Su,u).\]

Lemma: \(\sigma(S)\subset[m,M]\) and \(m,M\in\sigma(S)\).

Theorem: \(H\) a separable Hilbert space, \(S\) a symmetric compact operator on \(H\). Then there exists an ONB of \(H\) consisting of eigenvectors of \(S\).

Def:

  1. The formal adjoint of \(L\), denoted by \(L^*\), is

\[L^*v\equiv\sum_{ij}(a^{ij}v_{x_j})_{x_i}\]

  1. The adjoint bilinear form \(B^*:H_0^1\times H_0^1\rightarrow\mathbb{R}\) is \(B^*[v,u]\equiv B[u,v]\) for all \(u,v\in H_0^1\).

  2. If \(v\in H_0^1\) satisfies \(B^*[v,u]=(f,u)\) for all \(u\in H_0^1\) then it is a weak solution of the adjoint problem

\[\left\{\begin{matrix}L^*v=f & x\in U \\ v=0 & x\in\partial U.\end{matrix}\right.\]

\[\diamond\]

  • \(N=\{u\in H_0^1: u \text{ is a weak solution of the homogeneous problem} \}\)

  • \(N^*=\{v\in H_0^1: u \text{ is a weak solution of the homogeneous adjoint problem} \}\)

Theorem 4 (second existence theorem):

  1. Precisely one of the following

    1. for each \(f\in L^2\) there is a unique weak solution \(u\) of the BVP \((1)\)

    2. there exists a weak solution \(u\neq0\) of \((1)\) when \(f=0\) (homogeneous)

  2. If b. holds, \(\dim N=\dim N^*<\infty\)

  3. The BVP \((1)\) has a weak solution iff \((f,v)=0\) for all \(v\in N^*\).

\[\diamond\]

Theorem 5 (third existence theorem):

  1. There exists an at most countable set \(\Sigma\subset\mathbb{R}\) s.t. the BVP

\[\left\{\begin{matrix}Lu=\lambda u+f & x\in U \\ u=0 & x\in\partial U\end{matrix}\right.\]

has a weak solution for each \(f\in L^2\) iff \(\lambda\notin\Sigma\).

  1. If \(\Sigma\) is infinite, then \(\Sigma=\{\lambda_k\}\) is nondecreasing with \(\lambda_k\rightarrow\infty\).

Def: \(\Sigma\) is the real spectrum of \(L\). The BVP

\[\left\{\begin{matrix}Lu=\lambda u & x\in U \\ u=0 & x\in\partial U\end{matrix}\right.\]

has a nontrivial solution \(w\neq0\) iff \(\lambda\in\Sigma\). In this case \(\lambda\) is an eigenvalue of \(L\) corresponding to the eigenfunction \(w\). The PDE \(Lu=\lambda u\) for \(L=-\Delta\) is called Helmholtz’s equation.

Theorem 6 (boundedness of the inverse): If \(\lambda\notin\Sigma\) there exists a constant \(C\) s.t.

\[\|u\|_{L^2}\leq C\|f\|_{L^2}\]

whenever \(f\in L^2\) and \(u\in H_0^1\) is the unique weak solution of

\[\left\{\begin{matrix}Lu=\lambda u+f & x\in U \\ u=0 & x\in\partial U.\end{matrix}\right.\]

The constant \(C\) depends only on \(\lambda,U\) and the coefficients of \(L\). The constant \(C\) blows up when \(\lambda\) approaches an eigenvalue.

6.5 Eigenvalues and eigenfunctions

We consider

\[\begin{equation} \left\{\begin{matrix} Lw = \lambda w & x\in U \\ w = 0 & x\in\partial U, \end{matrix}\right. \end{equation}\]

with \(U\) open and bounded.

6.5.1 Eigenvalues of symmetric elliptic operators

Restrict attention to the divergence form \(Lu=-\sum_{ij}\left(a^{ij}u_{x_i}\right)_{x_j}\) with \(a^{ij}\in C^\infty(\bar U)\). Further suppose that the elliptic condition is satisfied and \(a^{ij}=a^{ji}\). Then the operator \(L\) is symmetric and, in particular, \(B[u,v]=B[v,u]\) (\(L\) is equal to its adjoint).

Assume also that \(U\) is connected.

Theorem 1 (eigenvalues of symmetric elliptic operators):

  1. each eigenvalue of \(L\) is real
  2. repeating each eigenvalue according to its multiplicity, we get \(\Sigma=\{\lambda_k\}\) s.t.

\[0<\lambda_1\leq\lambda_2\leq\cdots, \ \ \ \lambda_k\rightarrow\infty.\]

  1. There exists an ONB \(\{w_k\}\) of \(L^2(U)\), where \(w_k\in H_0^1\) is an eigenfunction corresponding to \(\lambda_k\):

\[\left\{\begin{matrix}Lw_k=\lambda_k w_k & x\in U \\ w_k=0 & x\in\partial U\end{matrix}\right., \ \ \ k=1,2,\dots\]

  • \(w_k\in C^\infty(U)\) and if \(\partial U\) is smooth, \(w_k\in C^\infty(\bar U)\).

Weyl’s law: The eigenvalues of the Laplacian in a smooth bounded open set \(U\subset\mathbb{R}\) satisfy

\[\lim_{k\rightarrow\infty}\frac{\lambda_k^{n/2}}{k}=\frac{(2\pi)^n}{|U|\alpha(n)}\]

where \(|U|\) is the volume of \(U\) and \(\alpha(n)=\frac{\pi^{n/2}}{\Gamma(\frac{n}{2}+1)}\) is the volume of a unit ball in \(\mathbb{R}^n\).

Def: We call \(\lambda_1>0\) the principle eigenvalue of \(L\).

Theorem 2 (variational principle for the principle eigenvalue):

  1. \(\lambda_1=\min\{B[u,u]:u\in H_0^1,\|u\|_{L^2}=1\}\)

  2. \(\lambda_1\) is obtained for a function \(w_1>0\) in \(U\) and solves

\[\left\{\begin{matrix}Lw_1=\lambda_1 w_1 & x\in U \\ w_1=0 & x\in\partial U\end{matrix}\right.\]

  1. if \(u\in H_0^1\) is any weak solution of

\[\left\{\begin{matrix}Lu=\lambda_1 u & x\in U \\ u=0 & x\in\partial U\end{matrix}\right.\]

then \(u\) is a multiple of \(w_1\).

  • remarks: assertion iii. states that \(\lambda_1\) has multiplicity one (\(0<\lambda<\lambda_2\leq\lambda_3\leq\dots\), its eigenspace is one-dimensional). the expression involving a minimum is called Rayleigh’s formula and is equivalent to

\[\lambda_1=\min_{\begin{matrix} u\in H_0^1 \\ u\neq0 \end{matrix}}\frac{B[u,u]}{\|u\|^2_{L^2}}.\]

6.5.2 Eigenvalues of nonsymmetric elliptic operators

Consider the nondivergence form \(Lu=-\sum_{ij}a^{ij}u_{x_ix_j}+\sum_ib^iu_{x_i}+cu\). Assume \(a^{ij},^i,c\in C^\infty(\bar U)\), \(U\) is open bounded and connected and \(\partial U\) is smooth. Suppose also that \(a^{ij}=a^{ji}\) and \(c\geq0\) in \(U\).

In this case \(L\) does not equal its adjoint.

Theorem 3 (principle eigenvalues for nonsymmetric elliptic operators):

  1. there exists a real eigenvalue \(\lambda_1\) for the operator \(L\), taken with zero boundary conditions, s.t. if \(\lambda\in C\) is any other eigenvalue, we have

\[\mathscr{R}(\lambda)\geq\lambda_1.\]

  1. there exists a corresponding eigenfunction \(w_1\) which is positive in \(U\).

  2. The eigenvalue \(\lambda_1\) is simple.

7.1 Second-order parabolic equations

7.1.1 Definitions

  • \(U\subset\mathbb{R}^n\), open and bounded

  • \(U_T=U\times[0,T]\), for some fixed \(T>0\).

\[\begin{equation} \left\{\begin{matrix} u_t+Lu=f & x\in U_T \\ u = 0 & x\in\partial U\times[0,T] \\ u = g & x\in U\times\{t=0\} \end{matrix}\right. \end{equation}\]

The coefficients \(a^{ij}\) and \(b^i\) of \(L\) are now functions of time as well as space. We call \(L\) parabolic if it is uniformly elliptic for each \(t\in[0,T]\).

  • \(\pmb u:[0,T]\rightarrow H_0^1(U)\) s.t. \([\pmb u(t)](x)\equiv u(x,t)\),  \(x\in U, t\in[0,T]\)

  • \(\pmb f:[0,T]\rightarrow L^2(U)\) s.t. \([\pmb f(t)](x)=f(x,t)\)

\[B[u,v;t]\equiv\int_U\left(\sum_{ij}a^{ij}(\cdot,t)u_{x_i}v_{x_j}+\sum_ib^i(\cdot)u_{x_i}v+c(\cdot,t)uv\right)dx, \ \ u,v\in H_0^1(U), \ \text{ ae } \ t\in [0,T]\]

Def: The function \(\pmb u\in L^2(0,T; H_0^1(U))\), with \(\pmb u'\in L^2(0,T;H^{-1}(U))\), is a weak solution of \((9)\) if

  1. \(\left<\pmb u',v\right>+B[\pmb u,v;t]=(\pmb f,v)\) for all \(v\in H^1_0(U)\) and a.e. \(t\in[0,T]\)

  2. \(\pmb u(0)=g\)

7.4 Semigroup theory

7.4.1 Definitions

\(X\) is a real Banach space. Consider the ODE

\[\left\{\begin{matrix}\pmb u'(t)=A\pmb u(t), & t\geq0 \\ \pmb u(0)=u, & u\in X \end{matrix}\right.\]

where \(A:D(A)\rightarrow X\) is a possibly unbounded operator whose domain is a linear subspace of \(X\). We consider the existence and uniqueness of a mapping \(\pmb u:[0,\infty)\rightarrow X\) that satisfies the above ODE (in the weak sense).

  • we keep \(X=L^p\) in mind. in this case \(A\) is neccessarily unbounded.

Notation: \(\pmb u(t)\equiv S(t)u, \ t\geq0\) to emphasize dependence on \(u\). \(S(t)\) is a mapping of \(X\) into \(X\).

  1. \(S(t):X\rightarrow X\) is linear

  2. \(S(0)u=u\)

  3. \(S(t+s)u=S(t)S(s)u=S(s)S(t)u\) (unique solution at each point)

  4. \(t\mapsto S(t)u\) is continuous from \([0,\infty)\) into \(X\)

Def:

  1. A family \(\{S(t)\}_{t\geq0}\) of bounded linear operators on \(X\) is a semigroup if a.-d. are satisfied.

  2. \(\{S(t)\}\) is a contraction semigroup if in addition \(\|S(t)\|\leq1\) where

\[\|S(t)\|\equiv\sup_{u\in X}\frac{\|S(t)u\|}{\|u\|}\]

Since \(\|S(t)u\|\leq\|S(t)\|\|u\|\), this implies \(\|S(t)u\|\leq\|u\|\).

  • NOTE: From here we assume that \(\{S(t)\}\) is a contraction semigroup

Def:

\[D(A)\equiv\left\{u\in X:\lim_{t\downarrow 0}\frac{S(t)u-u}{t}\text{ exists in }X\right\}\]

\[Au\equiv\lim_{t\downarrow 0}\frac{S(t)u-u}{t}, \ \ \ u\in D(A).\]

The operator \(A\) is the infinitesimal generator of the semigroup \(\{S(t)\}\).

Theorem 1 (differentiable properties of semigroups):

If \(u\in D(A)\) then

  1. \(S(t)u\in D(A)\) for each \(t\geq0\)
  2. \(AS(t)u=S(t)Au\) for each \(t\geq0\)
  3. \(t\mapsto S(t)u\) is differentiable for each \(t>0\).
  4. \(\frac{d}{dt}S(t)u=AS(t)u\) for \(t>0\)

Proof:

  1. & ii. Since \(u\in D(A)\), we know that

\[\lim_{t\downarrow0}\frac{S(t)u-u}{t}\equiv Au\in X.\]

Replacing \(u\) with \(S(t)u\) and applying the semigroup property (commutivity)

\[\lim_{s\downarrow0}\frac{S(s)S(t)u-S(t)u}{s}=\lim_{s\downarrow 0}\frac{S(t)(S(s)u-u)}{s}=S(t)Au.\]

Since \(A:D(A)\rightarrow X\) and \(S(t):X\rightarrow X\), we have \(S(t)A: D(A)\rightarrow X\) so the above limit exists in \(X\). Hence, \(S(t)u\in D(A)\) and thus it makes sense to write \(S(t)Au=AS(t)u\).

  1. & iv.

\[\lim_{h\downarrow0}\left\|\frac{S(t+h)u-S(t)u}{h}-AS(t)u\right\|=\lim_{h\downarrow0}\left\|\frac{S(t)S(h)u-S(t)u}{h}-AS(t)u\right\|\]

\[\leq\|S(t)\|\lim_{h\downarrow0}\left\|\frac{S(h)u-u}{h}-Au\right\|=0\]

Remark: The mapping \(t\mapsto S(t)\) is \(C^1\) in \((0,\infty)\) when \(u\in D(A)\).

Theorem 2 (properties of generators):

  1. \(D(A)\) is dense in \(X\)
  2. \(A\) is a closed operator

Remark: \(A\) is closed if whenever \(\{u_k\}\subset D(A)\) and \(u_k\rightarrow u\), \(Au_k\rightarrow v\), then

\[u\in D(A), \ \ v=Au.\]

Proof: exercise for bob

Resolvents

Let \(A\) be a closed linear operator on \(X\) with domain \(D(A)\).

Def:

  1. \(\lambda\in\rho(A)\) (\(\rho(A)\) is the resolvent set of \(A\)) if

\[\lambda I-A:D(A)\rightarrow X\]

is a bijection.

  1. If \(\lambda\in\rho(A)\), then \(R_\lambda:X\rightarrow X\) given by \(R_\lambda u\equiv(\lambda I-A)^{-1}u\) is called the resolvent operator of \(A\).

By the closed graph theorem, \(R_\lambda\) is a bounded linear operator. Also, \(R_\lambda\) and \(A\) commute.

Theorem 3 (properties of resolvent operators):

  1. if \(\lambda,\mu\in\rho(A)\), then \(R_\lambda\) and \(R_\mu\) commute.

  2. if \(\lambda>0\), then \(\lambda\in\rho(A)\) and

\[R_\lambda u=\int_0^\infty e^{-\lambda t}S(t)udt, \ \ \ u\in X.\]

Since

\[\|R_\lambda u\|=\left\|\int_0^\infty e^{-\lambda t}S(t)udt\right\|\leq\left|\int_0^\infty e^{-\lambda t}\|S(t)\|\|u\|dt\right|\]

\[\leq\|u\|\int_0^\infty e^{-\lambda t}dt=\frac{\|u\|}{\lambda}e^{-\lambda t}\]

we have \(\|R_\lambda\|\leq\frac{1}{\lambda}\) for all \(\lambda\in\rho(A)\cap(0,\infty)\).

Hence, the resolvent operator is the Laplace transform of the semigroup.

Generating contraction semigroups

Theorem 4 (Hill-Yosida) Let \(A\) be a closed, densely-defined linear operator on \(X\). Then \(A\) is the generator of a contraction semigroup \(\{S(t)\}\) iff

\[(0,\infty)\subset\rho(A), \ \ \ \|R_\lambda\|\leq\frac{1}{\lambda} \text{ for } \lambda>0.\]

Proof: later

Remark: Let \(\omega\in\mathbb{R}\). A semigroup \(\{S(t)\}\) is called \(\omega\)-contractive if \(\|S(t)\|\leq e^{\omega t}\) for all \(t\geq0\). A variant of theorem 4 states that a closed, densely-defined linear operator \(A\) generates an \(\omega\)-contractive semigroup iff

\[(\omega,\infty)\subset\rho(A), \ \ \ \|R_\lambda\|\leq\frac{1}{\lambda-\omega}\text{ for }\lambda>\omega\]
