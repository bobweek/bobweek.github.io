Stochastic partial differential equations

Motivation

Consider a biological population where each individual has a real valued trait \(z\). Let \(\mathfrak{n}_t(z)\) denote the abundance density of the trait value \(z\) at time \(t\). The total abundance of the population \(n_t\) is given by \(n_t=\int\mathfrak{n}_t(z)dz\). For each fixed \(z\in \mathbb{R}\) let \(\mathfrak{n}_t(z)\) be a real valued stochastic process satisfying the SDE

\[d\mathfrak{n}_t(z)=m(z)\mathfrak{n}_t(z)dt+\eta\mathfrak{n}_t(z)dB_t(z), \ \ \mathfrak{n}_0(z)>0.\]

The function \(m\in L^\infty(\mathbb{R})\) maps the trait value \(z\) to the Malthusian fitness of individuals with trait \(z\). The number \(0\leq\eta<\infty\) denotes the magnitude of demographic stochasticity and \(B_t(z)\) is a one dimensional standard Brownian motion for each fixed \(z\in\mathbb{R}\). For now we assume perfect heritability which may be implied by the more specific assumptions that the population is asexually reproducing and the effects of mutation and epigenetics are absent. This means the set \(\{\mathfrak{n}_t(z)\}_{z\in\mathbb{R}}\) consists of real valued independent stochastic processes.

The solution to \(d\mathfrak{n}_t(z)\) can be found in Øksendal (example 5.1.1) which is repeated here:

Fix \(z\in\mathbb{R}\) so that \(\mathfrak{n}_t(z)\) is viewed as a real valued stochastic process. The dependency on \(z\) is suppressed for clarity. Rearranging, we have

\[\frac{d\mathfrak{n}_t}{\mathfrak{n}_t}=mdt+\eta dB_t\]

and thus

\[\int_0^t\frac{d\mathfrak{n}_s}{\mathfrak{n}_s}ds=mt+\eta B_t.\]

Using the finite dimensional Itô formula on the function \(g(t,x)=\ln x, \ x>0\), we get

\[d(\ln \mathfrak{n}_t)=\frac{1}{\mathfrak{n}_t}d\mathfrak{n}_t+\frac{1}{2}\left(-\frac{1}{\mathfrak{n}_t}\right)(d\mathfrak{n}_t)^2\]

\[=\frac{d\mathfrak{n}_t}{\mathfrak{n}_t}-\frac{1}{2}\eta^2dt.\]

Combining these results, we find \(\mathfrak{n}_t=\mathfrak{n}_0\exp((m-\frac{1}{2}\eta^2)t+\eta B_t)\), which is a geometric Brownian motion (for each fixed \(z\in\mathbb{R}\)).

Since we assumed perfect heritability and that fitness for each trait is constant in time, the solution \(\mathfrak{n}_t(z)\) has been found for each \(z\). Thus, the total abundance is given as

\[n_t=\int_\mathbb{R}\mathfrak{n}_t(z)dz=\int_\mathbb{R}\mathfrak{n}_0(z)\exp((m(z)-\frac{1}{2}\eta^2)t+\eta B_t(z))dz.\]

Since \(\mathfrak{n}_t(z)>0\) for \(t\geq0, \ z\in\mathbb{R}\), we can use Tonelli’s theorem to justify

\[\mathbb{E}n_t=\int_\mathbb{R}\mathbb{E}[\mathfrak{n}_t(z)]dz.\]

Using the properties of geometric Brownian motion we know that, for each fixed \(z\in\mathbb{R}\) and \(t\geq0\), \(\mathfrak{n}_t(z)\) is a log-normal random variable with mean \(\mathbb{E}[\mathfrak{n}_t(z)]=e^{m(z)t}\mathbb{E}[\mathfrak{n}_0(z)]\). Writing \(p_t(z)=\mathfrak{n}_t(z)/n_t\), we have

\[\mathbb{E}n_t=\int_\mathbb{R}e^{m(z)t}\mathbb{E}[\mathfrak{n}_0(z)]dz=\int_\mathbb{R}e^{m(z)t}\mathbb{E}[n_0p_0(z)]dz.\]

Assuming the random number \(n_0\) and the random function \(p_0\) are independent, we have

\[\mathbb{E}n_t=\left(\mathbb{E}n_0\right)\int_\mathbb{R}e^{m(z)t}\mathbb{E}[p_0(z)]dz.\]

Carefully note that \(n_t\) is a geometric Brownian motion only when \(m(z)=m_0\in\mathbb{R}\) is constant with respect to \(z\). Hence, in general \(n_t\) does not share the same nice qualities as a geometric Brownian motion. It would be interesting to formulate and solve the SDE \(dn_t\) and the associated SPDE \(dp_t(z)\). This is especially true for \(dp_t(z)\) since in classical quantitative genetics \(p_t(z)\) is assumed to be a Gaussian probability density wrt \(z\). Developing an SPDE for \(dp_t(z)\) would relax this assumption while still capturing stochasticity derived from demography. However, it also requires us to calculate \(n_t\) from \(\mathfrak{n}_t(z)\) for which we need an appropriate chain rule. For functions on finite dimensionsional spaces, of course, this is given by Itô’s finite dimensional formula which can be found in almost any book on SDE. However, \(n_t\) is not a function on a finite dimensional space. To see this note that

\[\mathfrak{n}_0(z)\in L^1(\mathbb{R})\cap L^\infty(\mathbb{R})\]

since we require

\[\|\mathfrak{n}_0\|_1\equiv\int|\mathfrak{n}_0(z)|dz=n_0<\infty\]

and

\[\|\mathfrak{n}_0\|_\infty\equiv\sup_{z\in\mathbb{R}}|\mathfrak{n}_0(z)|<\infty.\]

Hence, Hölder’s inequality implies

\[\|\mathfrak{n}_0^2\|_1\leq\|\mathfrak{n}_0\|_1\|\mathfrak{n}_0\|_\infty<\infty.\]

In particular, \(\|\mathfrak{n}_0^2\|_1^{1/2}=\left(\int|\mathfrak{n}_0(z)|^2dz\right)^{1/2}=\|\mathfrak{n}_0\|_2<\infty\) so that \(\mathfrak{n}_0\in L^2(\mathbb{R})\). Because of this we can view \(n_t\) as a linear functional on \(L^2(\mathbb{R})\). Therefore, to calculate \(dn_t\) from \(\mathfrak{n}_t\) we will need an infinite dimensional extension of Itô’s formula.

Developing the theory for this extension will also allow us to work with actual SPDE where, for example, the set of SDE \(\{\mathfrak{n}_t(z)\}\) no longer evolve independently. In particular, we’ll be able to develop models accounting for imperfect heritability, sexual preference, frequency and density dependent selection, migration and so forth. If done correctly, I believe this approach will result in a beautiful synthesis of contemporary evolutionary and ecological theory. It is this advantage over previous treatments of evolutionary theory that make it worth pursuing, even if the road to this goal is lengthy and difficult.

Most of the material that follows comes from Stochastic equations in infinite dimensions by Da Prato & Zabczyk. I adapt their notation slightly to help clarify certain aspects (such as when a space needs to be a separable Hilbert space as opposed to just a Hilbert space) and rephrase some of their discussion in my own wording. However, some of the content is typed verbatim from their text. I also refer to two key papers by Curtain & Falb (Itô’s lemma in infinite dimensions & Stochastic differential equations in Hilbert space) for an alternative treatment of the infinite dimensional Itô formula and to Stochastic differential equations by Øksendal.

4.1 Wiener processes

  • Def 4.1: A real valued stochastic process \(W(t), \ t\geq0\), is called a Wiener process if

    1. \(W\) has continous trajectories and \(W(0)=0\).
    2. \(W\) has independent increments and

\[\mathscr{L}(W(t)-W(s))=\mathscr{L}(W(t-s)), \ t\geq s\geq0.\]

  1. \(\mathscr{L}(W(t))=\mathscr{L}(-W(t)), \ t\geq0. \\)
  • Def: (extension to a general linear (topological) space \(K\) with \(\sigma\)-algebra \(\mathscr{B}(K)\)) A \(K\)-valued stochastic process \(W(t), \ t\geq0\) satisfying 1.-3. is a Wiener process in \(K\).

4.1.1 Hilbert space valued Wiener processes

Let \(W\) be a \(U\)-valued Wiener process. Then \(\left<W(t),u\right>, \ t\geq0,\) is a real valued Wiener process and \(\mathscr{L}(W(t))\) is a Gaussian measure with mean 0. Furthermore, for any \(u,v\in U, \ t,s\geq0,\)

\[\mathbb{E}[\left<W(t),u\right>\left<W(s),u\right>]=(t\wedge s)\mathbb{E}[\left<W(1),u\right>^2]\]

and

\[\mathbb{E}[\left<W(t),u\right>\left<W(s),v\right>]=(t\wedge s)\mathbb{E}[\left<W(1),u\right>\left<W(1),v\right>]\]

\[=(t\wedge s)\left<Qu,v\right>\]

where \(Q\) is the covariance operator of the Gaussian measure \(\mathscr{L}(W(1))\). The operator \(Q\) is of trace class and for each distribution \(\mathscr{L}(W(1))\), there is a trace class \(Q\) that completely characterizes it.

  • Def 4.2: A \(U\)-valued stochastic process \(W(t), \ t\geq0\) is a \(Q\)-Wiener process if

    1. \(W(0)=0\).
    2. \(W\) has continuous trajectories.
    3. \(W\) has independent increments.
    4. \(\mathscr{L}(W(t)-W(s))=\mathscr{N}(0,(t-s)Q), \ t\geq s\geq0\). $$

Because of our assumptions about \(Q\) and since \(U\) has a complete orthonormal system \(\{e_k\}\subset U\), there exists a bounded sequence of nonnegative real numbers \(\{\lambda_k\}\) s.t. \(Qe_k=\lambda_ke_k, \ k\in\mathbb{N}\).

  • Proposition 4.3: If \(W(t)\) is a \(Q\)-Wiener process, then

    1. \(W\) is a Gaussian process on \(U\) and

\[\mathbb{E}(W(t))=0, \ \mathrm{Cov}(W(t))=tQ, \ t\geq0.\]

  1. For any \(t\geq0\), \(W\) has the expansion

\[W(t)=\sum_j\sqrt{\lambda_j}\beta_j(t)e_j\]

where

\[\beta_j(t)=\frac{1}{\sqrt{\lambda_j}}\left<W(t),e_j\right>, \ j\in\mathbb{N}\]

are real valued Brownian motions mutually independent on \((\Omega,\mathscr{F},\mathbb{P})\) and the series expansion of \(W(t)\) converges in \(L^2(\Omega,\mathscr{F},\mathbb{P})\).

  • BW: (comparison to KL expansion) The expansion above is strikingly remeniscient of the Karhunen-Loève (KL) expansion. However the above expansion is with respect to the Hilbert space in which \(W(t)\) takes its values. The KL expansion, on the other hand, is with respect to time. Specifically, the KL expansion is defined for a zero-mean process \(X(t)\in L^2(\Omega,\mathscr{F},\mathbb{P})\) with \(t\in[a,b]\) and continuous \(K(s,t)=\mathbb{E}(X(s)X(t))\). If \(\{\lambda_k\}\) denotes the set of eigenvalues of \(T:L^2([a,b])\rightarrow L^2([a,b])\) defined by \((Tf)(t)\equiv\int_a^b K(s,t)f(s)ds\), then the KL expansion of \(X(t)\) is

\[X(t)=\sum_j\sqrt{\lambda_j}\beta_je_j(t)\]

where

\[\beta_j=\frac{1}{\sqrt{\lambda_j}}\int_a^bX(t)e_j(t)dt, \ j\in\mathbb{N}\]

are uncorrelated zero-mean random variables with unit variance. The convergence of the KL expansion is in \(L^2(\Omega)\), uniform in \(t\). Notice that in Proposition 4.3, \(e_j\in U\) are not functions of time, but the \(\beta_j\) are. However, for the KL expansion the \(\beta_j\) are not functions of time but the \(e_j\) are.

4.1.2 Generalized Wiener processes over a Hilbert space

The primary reason for my study of SPDE is to formally treat PDE subjected to a specific kind of noise. In particular, the noise I wish to account for, denoted by \(\mathscr{W}\) where \(\mathscr{W}(x,t)\in\mathbb{R}\) for each \(x\in U\) (\(U\) is usually some subset of \(\mathbb{R}^n\)) and \(t\in[0,\infty)\), should satisfy the following properties:

  1. For every \(x\in U,s\leq t\in[0,\infty)\)

\[\mathscr{L}(\mathscr{W}(x,t)-\mathscr{W}(x,s))=\mathscr{L}(\mathscr{W}(x,t-s))=\mathscr{N}(0,t-s).\]

  1. For any \(x,y\in U\) and \(t,s\in[0,\infty)\)

\[\mathbb{E}(\mathscr{W}(x,t)\mathscr{W}(y,s))=(t\wedge s)\delta(x-y).\]

That is, \(\mathscr{W}(x,t)\) is a set of independent real valued Brownian motions indexed by \(x\in U\). Previous definitions of the Wiener process are insufficient to capture this since \(\mathscr{W}(t)=\{\mathscr{W}(x,t):x\in U\}\) is not valued in a convenient Hilbert space. In particular, my prefered Hilbert space is \(L^2(\mathbb{R}^n)\), but, even for \(n=1\), \(\int_\mathbb{R}\mathscr{W}(t,x)dx\) is not finite wp1! To circumvent this, Da Prato & Zabczyk give the following definition

  • Def: Any transformation that maps \(a\in\mathscr{U}\) to \(W_a\), a real valued Wiener processes on \([0,\infty)\), that is continuous in the sense

\[t\geq0, \{a_n\}\subset \mathscr{U}, \lim_{n\rightarrow\infty}\|a_n-a\|_{\mathscr{U}}=0\implies\lim_{n\rightarrow\infty}\mathbb{E}\left|W_a(t)-W_{a_n}(t)\right|^2=0.\]

is a generalized Wiener process over \(\mathscr{U}\).

  • BW: (remark) The integral transformation given by \(W_a(t)=\int_U a(x)\mathscr{W}(x,t)dx\) may qualify as such a mapping (for concreteness, I think of \(U=\mathbb{R}^n\) and \(\mathscr{U}=L^2(\mathbb{R}^n)\)).

This definition implies the existence of a bilinear form \(K(a,b), \ a,b\in \mathscr{U}\) s.t.

\[\mathbb{E}[W_a(t)W_b(s)]=(t\wedge s) \ K(a,b), \ \ t,s\geq0, \ a,b\in\mathscr{U}.\]

The continuity condition gurantees that \(K\) is continuous in \(\mathscr{U}\) and hence there exists \(\mathscr{Q}\in L(\mathscr{U})\) s.t.

\[\mathbb{E}[W_a(t)W_b(s)]=(t\wedge s)\left<\mathscr{Q}a,b\right>, \ \ t,s\geq0, \ a,b\in\mathscr{U}.\]

This operator is self-adjoint and positive definite and is the covariance of the generalized Wiener process \(\mathscr{W}.\)

If \(\mathscr{Q}\) is the identity operator, so that \(\mathscr{Q}a=a, \ a\in \mathscr{U}\), then \(\mathscr{W}\) is called a cylindrical Wiener process in \(\mathscr{U}\). As mentioned in the primer on SPDE by Martin Hairer, it is unfortunate that \(\mathscr{W}\) is referred to as a process in \(\mathscr{U}\) since it is not \(\mathscr{U}\)-valued. Hairer suggests that \(\mathscr{W}\) should be called a cylindrical Wiener process over \(\mathscr{U}\).

  • BW: (false issue) At first glance, I feel there is an issue here. For my process \(\mathscr{W}\) introduced at the beginning of this section, I mistakenly believe I need \(K(a,b)=\delta(a-b)\). Thus, the above result is a generalization of condition (2.) on the process \(B\) where \(\mathbb{E}[\mathscr{W}(t,y)\mathscr{W}(s,z)]=(t\wedge s)\delta(y-z)\). Hence, everything is just fine.

  • Example: Let \(\mathscr{W}\) be a cylindrical Wiener process over \(\mathscr{U}=L^2(\mathbb{R})\). Then for any \(u,v\in L^2(\mathbb{R})\)

\[\mathbb{E}[W_u(t)W_v(s)]=(t\wedge s)\int_\mathbb{R}u(x)v(x)dx.\]

This is the case that I will pay special attention to.

If \(\mathscr{U}_0=\mathscr{Q}^{1/2}(\mathscr{U})\) we call \(\mathscr{U}_0\) the reproducing kernel of the generalized Wiener process.

  • Constructing a generalized Wiener process from an arbitrary self-adjoint and positive definite operator \(\mathscr{Q}\):

Let \(\{e_j\}\) be a complete orthonormal basis in \(U_1\) and \(\{\beta_j\}\) a sequence of independent real valued standard Wiener processes. Define

\[W_a(t)\equiv\sum_j\left<\mathscr{Q}^{1/2}e_j,a\right>\beta_j(t), \ \ t\geq0, \ a\in\mathscr{U}.\]

  • Proposition 4.8: For every \(a\in\mathscr{U}\) we have

\[W_a(t)=\sum_j\left<\mathscr{Q}^{1/2}e_j,a\right>W_{\mathscr{Q}^{1/2}e_j}(t).\]

This forms a generalized Wiener process with covariance \(\mathscr{Q}\).

  • Intuitive result: An arbitrary generalized Wiener process over \(\mathscr{U}\) is a classical Wiener process taking values in some larger Hilbert space \(\mathscr{U}'\).

  • BW: Interpreting this result, if \(\mathscr{U}=L^2(\mathbb{R})\), we could take \(\mathscr{U}'=L^2_\rho(\mathbb{R})\) which is the set of all real valued functions \(u\) over \(\mathbb{R}\) s.t. \(\int_\mathbb{R}|u(x)|^2\rho(x)dx<\infty\) for some nonnegative fixed \(\rho\in L^2(\mathbb{R})\). Clearly \(L^2_\rho=L^2(\mathbb{R},\mathscr{B}(\mathbb{R}),\nu)\) where the measure \(\nu\) is given by \(\nu(A)=\int_A\rho(x)dx\). It is also clear that \(L^2(\mathbb{R})\subset L^2_\rho(\mathbb{R})\) since for \(v\in L^2(\mathbb{R})\), we have \(w=|v|^2\in L^2\) and hence the Schwartz inequality implies \(\int_\mathbb{R}|v(x)|^2\rho(x)dx\leq\|w\|_2\|\rho\|_2<\infty\).

If we set \(\rho\) s.t. \(\int\rho^2=1\) then we can think of a cylindrical Wiener process \(\mathscr{W}\) over \(L^2(\mathbb{R})\) (so that \(\mathscr{W}\) takes values in \(L^2_\rho(\mathbb{R})\)) as a function of time which, for each \(t\), returns an infinite set of mean zero normal random variables with variance \(t\) that we can “sample” using \(\rho\). That is,

\[\int_\mathbb{R}\mathscr{W}(t,x)\rho(x)dx=X(t)\]

is a well defined random variable for each \(t\). Partly by definition of \(\mathscr{W}\) it seems obvious that if \(\rho^2\) is a probability density function, then \(X(t)\) is a normally distributed random variable with mean zero and variance \(t\). In fact, since \(\mathbb{E}|W(x,t)|\) is finite for each \(x\in\mathbb{R}\) and \(t\geq0\), we have

\[\mathbb{E}\int_\mathbb{R}|\mathscr{W}(t,x)||\rho(x)|dx\leq\int_\mathbb{R}\rho(x)\mathbb{E}|\mathscr{W}(x,t)|dx<\infty\]

and therefore we may swap the order of Lebesgue integration to show

\[\mathbb{E}X(t)=0.\]

Note also that, for a fixed \(t\), \(\mathscr{W}(t,x)\) is white-noise wrt \(x\). That is, \(\mathscr{W}(t,x)\) is, in some sense, the derivative of a Brownian motion over \(x\in\mathbb{R}\). Restricting our attention to the closed subset \([a,b]\subset\mathbb{R}\) we then have

\[\int_a^b\mathscr{W}(t,x)dx\]

is a mean zero normally distributed random variable with variance \((b-a)t\). Hence, by partitioning \(\mathbb{R}\) into a finite set of intervals \(\{[a_i,b_i]\}\) we can define \(\rho\) as a simple function so that it is constant on each interval and

\[\int_\mathbb{R}\rho^2=\sum_i\rho_i^2(b_i-a_i)=1.\]

With \(\rho\) defined in this manner we have

\[\mathrm{Var}(X(t))=\mathrm{Var}\left(\sum_i\int_{I_i}\mathscr{W}(t,x)\rho(x)dx\right)\]

\[=\sum_i\rho_i^2\mathrm{Var}\left(\int_{a_i}^{b_i}\mathscr{W}(t,x)dx\right)=\sum_i\rho_i^2(b_i-a_i)t=t.\]

For the final step we simply note that any nonnegative measurable function can be approximated arbitrarily closely by a simple function. Using Lebesgue’s dominated convergence theorem, we achieve our result.

Of course, this can be applied to nonnegative \(\rho\in L^2(\mathbb{R})\) where \(\int\rho^2\neq1\). In this case, we simply get

\[\mathrm{Var}(X(t))=\|\rho\|_2^2t.\]

  • Example 4.9: (Brownian sheets) Let \(\mathscr{O}=[0,\pi]\) and \(g(x,y)=x\wedge y, \ x,y\in[0,\pi]\). Note that \(g\) is continuous and, since for a one dimensional Wiener process \(\beta\)

\[x\wedge y=\mathbb{E}[\beta(x)\beta(y)]\]

\(g\) is positive definite. The eigenvalues and eigenfunctions of the corresponding \(Q\) are

\[\lambda_n=(n+\frac{1}{2})^{-2}, \ e_n=\sqrt{\frac{2}{\pi}}\sin\left[\left(n+\frac{1}{2}\right)x\right], \ x\in[0,\pi], \ n\in\mathbb{N}\]

hence

\[W(t,x)=\sum_n\frac{\beta_n(t)}{n+\frac{1}{2}}\sqrt{\frac{2}{\pi}}\sin\left[\left(n+\frac{1}{2}\right)x\right], \ t\geq0, \ x\in[0,\pi]\]

where the \(\beta_n\) are independent real standard Brownian motions. From this we have

\[\mathbb{E}[W(t,x)W(s,y)]=(t\wedge s)(x\wedge y).\]

Fields \(W\) with the above covariance are called Brownian sheets on \([0,\infty)\times[0,\pi]\).

is positive definite on \(\mathscr{O}_1\times\mathscr{O}_2\).

  • Example 4.12: (Brownian sheets on \([0,\infty)\times\mathscr{O}\)) Let \(\mathscr{O}\subset\mathbb{R}^n\) and \(g\) a positive definite function on \(\mathscr{O}\). By prop 4.11

\[(t\wedge s)g(x,y), \ (t,x),(s,y)\in[0,\infty)\times\mathscr{O},\]

is also positive definite and there exists a Gaussian family \(W(t,x), \ t\geq0, \ x\in\mathscr{O}\) s.t.

\[\mathbb{E}[W(t,x)W(s,y)]=(t\wedge s)g(x,y).\]

For each \(x\), \(W(\cdot,x)\) has all the properties of a real valued Wiener process except for the regularity of trajectories. To prove the existence of a proper modification requires additional properties of \(g\).

4.1.4. Spatially homogeneous Wiener processes

  • This section is included becuase it provides and explicit treatment of a cylindrical Wiener process on \(L^2(\mathbb{R}^n)\).

  • Def: If \(\mu\) is a probability measure on \((E,\mathscr{B}(E))\), then the function \(\hat\mu\) on \(E^*\)

\[\hat\mu(y)\equiv\int_Ee^{iy(x)}\mu(dx), \ \ y\in E^*\]

is the characteristic function of \(\mu\). If \(E=H\) is a Hilbert space then

\[\hat\mu(\lambda)=\int_He^{i\left<\lambda,x\right>}\mu(dx), \ \ \lambda\in H.\]

  • Bochners theorem: If \(\phi:U\rightarrow\mathbb{C}\) is a characteristic function of a probability measure \(\mu\) on \((U,\mathscr{B}(U))\) then

    1. \(\phi\) is a continuous, positive definite function s.t. \(\phi(0)=1\).
    2. For any \(\epsilon>0\) there exists a nonnegative nuclear operator \(S_\epsilon\) s.t.

\[1-\Re\phi(\lambda)\leq\epsilon, \ \ \forall \ \lambda:\left<S_\epsilon\lambda,\lambda\right>\leq1.\]

Conversly, if \(\phi\) satisfies 1. & 2., then it is a characteristic function of some probability measure on \(U\).

Let \(\mu\) be a finite nonnegative measure on \(\mathbb{R}^n\) and

\[\Gamma(\lambda)\equiv\int_{\mathbb{R}^n}e^{i\left<\lambda,x\right>}\mu(dx), \ \lambda\in\mathbb{R}^n,\]

then \(g(x,y)=\Gamma(x-y), \ x, \ y\in\mathbb{R}^n,\) is a continuous, bounded, positive definite function. Under some further (mild) conditions on \(\mu\) one can show there exists a continuous random field \(W(t,x)\) s.t. \(W(t,\cdot)\) is a Wiener process in \(L_\rho^2(\mathbb{R}^n)\), for an integrable weight \(\rho\), s.t.

\[\mathbb{E}[W(t,x)W(s,y)]=(t\wedge s)\Gamma(x-y).\]

Then

\[\mathbb{E}\|W(t,\cdot)\|_{L_\rho^2}^2=\mathbb{E}\int_{\mathbb{R}^n}|W(t,x)|^2\rho(x)dx\]

\[=\int_{\mathbb{R}^n}\mathbb{E}|W(t,x)|^2\rho(x)dx=t\Gamma(0)\int_{\mathbb{R}^n}\rho(x)dx.\]

We now generalize. Let \(\mathscr{S}_c(\mathbb{R}^n)\) denote the space of \(\mathbb{C}\)-valued infinitely differentiable functions \(\psi\) on \(\mathbb{R}^n\) for which each seminorm

\[|\psi|_{\alpha,\beta}=\sup_{x\in\mathbb{R}^n}|x^\alpha D^\beta\psi(x)|\]

is finite. The dual space \(\mathscr{S}^*_c(\mathbb{R}^n)\) is the space of tempered distributions. The space \(\mathscr{S}(\mathbb{R}^n)\) consists of real valued functions from \(\mathscr{S}_c(\mathbb{R}^n)\). The space \(\mathscr{S}^*(\mathbb{R}^n)\) consists of real valued functionals from \(\mathscr{S}^*_c(\mathbb{R}^n)\). We suppress the dependency on \(\mathbb{R}^n\).

For \(\psi\in\mathscr{S}_c\) define \(\psi_{(s)}\equiv\overline{\psi(-x)}, \ x\in\mathbb{R}^n.\) Denote \(\mathscr{S}_{(s)}=\{\psi\in\mathscr{S}_c:\psi=\psi_{(s)}\}\) and \(\mathscr{S}_{(s)}^*=\{u\in\mathscr{S}^*:(u,\psi)=(u,\psi_{(s)}s) \ \forall \ \psi\in\mathscr{S}_{(s)}\}\). For \(\psi\in\mathscr{S}_c\) write \(\hat\psi\) for the Fourier transform

\[\hat\psi(\lambda)=\left(\frac{1}{2\pi}\right)^{n/2}\int_{\mathbb{R}^n}e^{-2\pi i\left<\lambda,x\right>}\psi(x)dx.\]

For each \(u\in\mathscr{S}_c^*\), \(\hat u\) is given by \((\hat u,\hat\psi)=(u,\psi)\).

The distribution \(\Gamma\in\mathscr{S}^*\) is positive definite if for each \(\psi\in\mathscr{S}\)

\[(\Gamma,\psi*\psi_{(s)})\geq0.\]

  • Theorem 4.13: A distribution \(\Gamma\in\mathscr{S}^*\) is positive definite if there exists a measure \(\mu\in\mathscr{S}^*\) symmetric around 0 s.t. \(\Gamma=\hat\mu\). Then \(\mu\) is called the spectral measure of \(\Gamma\).

  • Def: An \(\mathscr{S}^*\)-valued continuous stochastic process \(\mathscr{W}\) is called a Wiener process with covariance kernel \(\Gamma\) if

    1. For each \(\psi\in\mathscr{S}\) the process \((\mathscr{W},\psi)\) is a real valued Wiener process
    2. For all \(s,t\geq0,\phi,\psi\in\mathscr{S}\) we have

\[\mathbb{E}[(\mathscr{W}(t),\phi)(\mathscr{W}(s),\psi)]=(t\wedge s)(\Gamma,\phi*\psi_{(s)}).\]

  • Theorem 4.14: For any positive definite distribution \(\Gamma\) there exists a Wiener process \(W\) with covariance kernel \(\Gamma\). Furthermore, there is a Hilbert space \(U_\Gamma\), continuously embedded in \(\mathscr{S}^*\), s.t.

\[\mathscr{L}((\mathscr{W}(t),\psi))=\mathscr{N}(0,t\|\psi\|^2_{U_\Gamma}), \ \ \forall \ \psi\in\mathscr{S}.\]

  • Example 4.16: Let \(\mu=\delta_0\), the delta measure at 0, then

\[\Gamma=\hat\delta_0=\left(\frac{1}{2\pi}\right)^{n/2}\lambda_n,\]

where \(\lambda_n\) denotes the Lebesgue measure on \(\mathbb{R}^n\). The space \(U_\Gamma\) is then one dimensional and consists of constant functions (so is it isomorphic to \(\mathbb{R}\)?). The process \(\mathscr{W}\) is then of the form

\[\mathscr{W}=\left(\frac{1}{2\pi}\right)^{n/2}\beta(t)\lambda_n\]

where \(\beta(t)\) is a standard Brownian motion.

  • Example 4.17: Let \(\mu=\lambda_n\), then

\[\Gamma=\hat\lambda_n=(2\pi)^{n/2}\delta_0.\]

For any \(\phi,\psi\in\mathscr{S}\) we then have

\[(\Gamma,\phi*\psi_{(s)})=(2\pi)^{n/2}\int_{\mathbb{R}^n}\phi(-x)\psi_{(s)}(x)dx\]

\[=(2\pi)^{n/2}\int_{\mathbb{R}^n}\phi(-x)\psi(-x)dx=(2\pi)^{n/2}\int_{\mathbb{R}^n}\phi(x)\psi(x)dx\]

Hence, if \(\mathscr{W}=(2\pi)^{-n/2}\beta(t)\delta_0\) then \((2\pi)^{n/2}\mathscr{W}\) is a cylindrical Wiener process on \(L^2(\mathbb{R}^n)\).

4.1.5 Complements on a Brownian sheet

  • The point of including this section is that it connects Brownian sheets on \([0,\infty)\times\mathbb{R}^n\) (as developed in 4.1.3) with cylindrical Wiener processes on \(L^2(\mathbb{R}^n)\) (as developed in 4.1.4).

This section shows that the distributional derivative of a Brownian sheet \(B\) on \([0,\infty)\times\mathbb{R}^n\) is “eqaul” to the cylindrical Wiener process \(\mathscr{W}\) on \(L^2(\mathbb{R}^n)\). That is

\[\frac{\partial^n}{\partial x_1\cdots\partial x_n}B(t,x)=\mathscr{W}(t,x), \ x\in\mathbb{R}^n.\]

  • Def: A Brownian sheet \(B\) on \([0,\infty)\times[0,\infty)^n\) is a Gaussian random field with correlation function

\[(t\wedge s)\prod_j(x_j\wedge y_j), \ \ t,s,x_j,y_j\geq0.\]

Now let \(B_l\) be a family of independent Brownian sheets on \([0,\infty)\times[0,\infty)^n\) indexed by the family of all sequences \(l=\{l_i\}\) where each \(l_i\in\{0,1\}\). Then, for any \(t\geq0\) and \(x=(x_1,\dots,x_n)\in\mathbb{R}^n\), we define

\[B(t,x)\equiv B_l(t,(-1)^{l_1}x_1,\dots,(-1)^{l_n}x_n)\]

when each \((-1)^{l_i}x_i\geq0\). The resulting object \(B\) is called the Brownian sheet on \([0,\infty)\times\mathbb{R}^n\). The Kolmogorov continuity test allows us to assume each realization of \(B\) is continuous. Furthermore, \(B\) can be thought of as an \(L^2_\rho(\mathbb{R})\)-valued Wiener process with weight \(\rho\) s.t.

\[\int_{\mathbb{R}^n}|x_1|\cdots|x_n|\rho(x_1,\dots,x_n)dx_1\cdots dx_n<\infty.\]

  • Proposition 4.19: If \(B\) is a Brownian sheet on \([0,\infty)\times\mathbb{R}^n\) then, for any \(\phi,\psi\in\mathscr{S}(\mathbb{R}^n)\),

\[\mathbb{E}\left(\frac{\partial^n}{\partial x_1\cdots\partial x_n}B(t),\phi\right)\left(\frac{\partial^n}{\partial x_1\cdots\partial x_n}B(t),\psi\right)=(t\wedge s)\int_{\mathbb{R}^n}\phi(x)\psi(x)dx.\]

4.2 Definition of the stochastic integral

The development of the infinite dimensional stochastic integral is analogous to the development of the stochastic integral for finite dimensional stochastic processes. We begin with a definition for elementary processes and progressively extend this definition until it is sufficiently general.

Consider the \(U\)-valued \(Q\)-Wiener process \(W(t)\) in \((\Omega,\mathscr{F},\mathbb{P})\) and the normal filtration \(\{\mathscr{F}_t\}_{t\geq0}\) in \(\mathscr{F}\). Assume that

  1. \(W(t)\) is \(\mathscr{F}_t\)-measurable
  2. \(W(t+h)-W(t)\) is independent of \(\mathscr{F}_t, \ \ \ \ \forall \ h\geq0, \ \ \forall \ t\geq0.\)

Then \(W(t)\) is called a \(Q\)-Wiener process with respect to \(\{\mathscr{F}_t\}_{t\geq0}\). Fix \(T<\infty\).

  • Def: An \(L=L(U,H)\)-valued process \(\Phi(t), \ t\in[0,T]\) taking only a finite number of values is said to be elementary if \(\exists\) a partition \(0=t_0<\dots<t_k=T\) and a sequence \(\Phi_0,\dots,\Phi_{k-1}\) of \(L\)-valued random variables s.t. the \(\Phi_m\) are \(\mathscr{F}_{t_m}\)-measurable and bounded (in the sense that \(\|\Phi_m\|_L\) is a bounded random variable for each \(m\)) and \(\Phi(t)=\Phi_m\) for \(t\in(t_m,t_{m+1}]\), \(m=0,\dots,k-1\). For elementary processes \(\Phi\) the stochastic integral is defined by

\[\int_0^t\Phi(s)dW(s)\equiv\sum_{m=0}^{k-1}\Phi_m(W_{t_{m+1}\wedge t}-W_{t_m\wedge t})\]

and is denoted by \(\Phi\cdot W(t), \ t\in[0,T]\).

  • Comparison to SDE: The only difference is that \(W(t)\) is a real valued standard Wiener process (so that \(Q=1\)) and the \(\Phi_m\) are real valued random variables.

The subspace \(U_0\equiv Q^{1/2}(U)\) of \(U\), endowed with the inner product

\[\left<u,v\right>_0\equiv\sum_k\frac{1}{\lambda_k}\left<u,e_k\right>\left<v,e_k\right>=\left<Q^{-1/2}u,Q^{-1/2}v\right>, \ \ u,v\in U_0\]

is a Hilbert space. The space of all Hilbert-Schmidt operators \(L_2^0\equiv L_2(U_0,H)\) is a separable Hilbert space with the norm

\[\|\Psi\|_{L_2^0}^2\equiv\sum_{h,k}\left|\left<\Psi \sqrt{\lambda_h}e_h,f_k\right>\right|^2=\sum_{h,k}\lambda_h|\left<\Psi e_h,f_k\right>|^2\]

where \(\{\sqrt{\lambda_j}e_j\}\), \(\{e_j\}\) and \(\{f_j\}\) are complete orthonormal bases in \(U_0\), \(U\) and \(H\) respectively. Let \(\Phi(t), \ t\in[0,T]\) be a measurable \(L_2^0\)-valued process. Define the norms

\[\|\Phi\|_t\equiv\left[\mathbb{E}\int_0^t\|\Phi(s)\|_{L_2^0}^2ds\right]^{1/2}\]

\[=\left[\mathbb{E}\int_0^t\mathrm{Tr}[(\Phi(s)Q^{1/2})(\Phi(s)Q^{1/2})^*]ds\right]^{1/2}, \ \ t\in[0,T].\]

  • Proposition 4.20: If a process \(\Phi\) is elementary and \(\|\Phi\|_T<\infty\) then the process \(\Phi\cdot W\) is a contiunous square integrable \(H\)-valued martingale on \([0,T]\) and

\[\mathbb{E}|\Phi\cdot W(t)|^2=\|\Phi\|_T^2, \ \ 0\leq t\leq T.\]

  • BW (Comparison to SDE): This is analogous to the Itô isometry which states that if \(\Phi(t)\) is elementary in the sense of SDE and if there exists \(M>0\) s.t. \(|\Phi_m(\omega)|<M\) for each \(\omega\in\Omega, \ m=0,\dots,k-1\), then

\[\mathbb{E}\left[\left(\int_0^T\Phi(t)dW(t)\right)^2\right]=\mathbb{E}\left(\int_0^T[\Phi(t)]^2dt\right).\]

Since \(\mathbb{R}\)

  • Remark 4.21: The stochastic integral is a linear transformation from the space of all elementary process with the norm \(\|\cdot\|_T\) into the space \(\mathscr{M}_T^2(H)\) of \(H\)-valued martingales.

The integrands of stochastic intregrals can be regarded as random variables defined on \(\Omega_\infty\equiv[0.\infty)\times\Omega\) (or \(\Omega_T\equiv[0,T]\times\Omega\)) with the \(\sigma\)-field \(\mathscr{B}([0,\infty))\times\mathscr{F}\) (or \(\mathscr{B}([0,T])\times\mathscr{F}\)). The product of the Lebesgue measure on \([0,\infty)\) (or, \([0,T]\)) and the probability measure \(\mathbb{P}\) is denoted by \(\mathbb{P}_\infty\) (or \(\mathbb{P}_T\)).

Taking into account the adaptability of processes we choose the \(\sigma\)-field generated by the adapted simple process (\(\mathscr{P}_\infty\) or \(\mathscr{P}_T\)). The class of processes proper as integrands of stochastic integrals are predictable processes with values in \(L_2^0\). That is, they are measurable mappings from \((\Omega_\infty,\mathscr{P}_\infty)\) (or \((\Omega_T,\mathscr{P}_T)\)) into \((L_2^0,\mathscr{B}(L_2^0))\).

  • Proposition 4.22:

    1. If a mapping \(\Phi\) from \(\Omega_T\) into \(L\) is \(L\)-predictable then \(\Phi\) is also \(L_2^0\)-predictable. Hence, elementary processes are \(L_2^0\)-predictable

    2. If \(\Phi\) is an \(L_2^0\)-predictable process s.t. \(\|\Phi\|_T<\infty\) then there exists a sequence of elementary processes \(\{\Phi_n\}\) s.t. \(\|\Phi-\Phi_n\|_T\rightarrow0\) and \(n\rightarrow\infty\).

This allows us to extend the stochastic integral to any \(L_2^0\)-predictable process \(\Phi\) s.t. \(\|\Phi\|_T<\infty\). These processes form a Hilbert space denoted \(\mathscr{N}_W^2(0,T;L_2^0)\) (or just \(\mathscr{N}_W^2(0,T)\), or even \(\mathscr{N}_W^2\)) and by the above proposition, the elementary processes form a dense set in \(\mathscr{N}_W^2\). From proposition 4.20, the stochastic integral \(\Phi\cdot W\) is an isometry from that dense set into \(\mathscr{M}_T^2(H)\). Hence, the definition of the integral ca be extended to all of \(\mathscr{N}_W^2\). Furthermore, \(\mathbb{E}|\Phi\cdot W(t)|^2=\|\Phi\|_t^2, \ 0\leq t\leq T\) holds true. As a final step, the integral is extended one step further to \(L_2^0\)-predictable processes satisfying the even weaker condition

\[\mathbb{P}\left(\int_0^T\|\Phi(s)\|^2_{L_2^0}ds<\infty\right)=1.\]

Any such process is called stochastically integrable on \([0,T]\). They form a linear space denoted by \(\mathscr{N}_W(0,T;L_2^0\) (or \(\mathscr{N}_W(0,T)\), or even \(\mathscr{N}_W\)). The extension is carried by the localization procedure.

  • Lemma 4.24: Assume that \(\Phi\in \mathscr{N}_W^2\) and \(\tau\) is an \(\mathscr{F}_\tau\)-stopping time s.t. \(\mathbb{P}(\tau\leq T)=1\). Then

\[\int_0^tI_{[0,\tau]}(s)\Phi(s)dW(s)=\Phi\cdot W(\tau\wedge t), \ \ \ t\in (0,T], \ \mathbb{P}\text{-a.s.}\]

Assuming the above weak condition, define

\[\tau_n\equiv\inf\left\{t\in[0,T]:\int_0^t\|\Phi(s)\|_{L_2^0}^2ds\geq n\right\}\]

with the convention that the infimum of the empty set returns \(T\). Then \(\{\tau_n\}\) is a sequence s.t.

\[\mathbb{E}\int_0^t\|I_{[0,\tau_n)}(s)\Phi(s)\|_{L_2^0}^2ds<\infty.\]

This implies that stochastic integrals \(I_{[0,\tau_n]}\Phi\cdot W(t), \ t\in[0,T]\) are well defined for each \(n\). If \(n<m\) the \(\mathbb{P}\)-a.s.

\[I_{[0,\tau_n]}\Phi\cdot W(t)=(I_{[0,\tau_n]}(I_{[0,\tau_m]}\Phi)\cdot W(t))=(I_{[0,\tau_m]}\Phi)\cdot(\tau_n\wedge t), \ \ t\in [0,T].\]

Hence, one can assume the previous expectation holds for all \(\omega\in\Omega, \ t\in[0,T], \ n<m\). For any \(t\in[0,T]\) define

\[\Phi\cdot W(t)\equiv I_{[0,\tau_n]}\Phi\cdot W(t)\]

where \(n\) is an arbitrary natural number s.t. \(\tau_n\geq t\). If also \(\tau_m\geq t\) and \(m>n\) then

\[(I_{[0,\tau_m]}\Phi)\cdot W(t)=(I_{[0,\tau_m]}\Phi)\cdot W(t)(\tau_n\wedge t)=I_{[0,\tau_n]}\Phi\cdot W(t)\]

and thus the above definition is consistent.

4.4 The Itô formula

  • Verbatim from the text:

Assume that \(\Phi\) is an \(L_2^0\)-valued process stochastically integrable in \([0,T]\), \(\phi\) a \(H\)-valued predictable process Bochner integrable on \([0,T]\), \(\mathbb{P}\)-a.s., and \(X(0)\) a \(\mathscr{F}_0\)-measurable \(H\)-valued random variable. Then the following process

\[X(t)=X(0)+\int_0^t\phi(s)ds+\int_0^t\Phi(s)dW(s), \ t\in[0,T]\]

is well defined. Assume that a function \(F:[0,T]\times H\rightarrow\mathbb{R}^1\) and its partial derivatives \(F_t,F_x,F_{xx}\), are uniformly continuous on bounded subsets of \([0,T]\times H\).

  • Theorem 4.32: Under the above conditions, \(\mathbb{P}\)-a.s., for all \(t\in[0,T]\)

\[F(t,X(t))=F(0,X(0))+\int_0^t\left<F_x(s,X(s)),\Phi(s)dW(s)\right>\]

\[+\int_0^t\Big{\{}F_t(s,X(s))+\left<F_x(s,X(s)),\phi(s)\right>\]

\[\left.+\frac{1}{2}\mathrm{Tr}[F_{xx}(s,X(s))(\Phi(s)Q^{1/2})(\Phi(s)Q^{1/2})^*]\right\}ds\]

  • BW: (interpretation) The process \(X(t)\) is some \(H\)-valued process. For concreteness \(H\) might be the familiar \(\mathbb{R}^n\) or \(L^2(\mathbb{R}^n)\). However, recall the cylindrical Wiener process takes values in a “higher” Hilbert space (such as \(L^2_\rho(\mathbb{R}^n)\)). So if \(X(t)\) belongs to one of our familiar Hilbert spaces, but we’re integrating against a cylindrical Wiener process, we need some transformation \(\Phi\) that maps the “misbehaved” cylindrical Wiener process into something more manageable (something \(H\)-valued in particular). This way both integrals above return \(H\)-valued processes and, thus, their sum is also an \(H\)-valued process. This is consistent with the statement that \(X(t)\) is \(H\)-valued. For each \(t\), the function \(F(t)\) takes values from \(H\) and maps them to \(\mathbb{R}\) in a sufficiently smooth manner. This allows us to use the “change of variables” formula above (Itô’s formula). Note that the inner product in this formula is with respect to \(H\).

  • BW: (example) Let \(B\) be a Brownian sheet on \([0,\infty)\times\mathbb{R}\). Then, obviously, \(B(t)=\int_0^tdB(s)\). Given a nonnegative \(\psi\in L^2(\mathbb{R})\) define \(F:L_\psi^2(\mathbb{R})\rightarrow \mathbb{R}\) by \(F(u)=\int_\mathbb{R}\psi(x)u(x)dx\). We should get

\[W(t)=F(B(t))=\int_\mathbb{R}\psi(x)B(t,x)dx,\]

a one dimensional Wiener process. This is obviously the case from the above definition of a Brownian sheet. However, for the sake of applying Itô’s formula, we note \(F_t=0\) and set \(\|v\|_{L_\psi^1}\equiv\int v(x)\psi(x)dx\) so that \(F_u, F_{uu}\) are given by

\[F_u(v)\equiv \lim_{\epsilon\rightarrow0}\frac{1}{\|\epsilon v\|_{L_\psi^1}}(F(u+\epsilon v)-F(u))\]

\[=\lim_{\epsilon\rightarrow0}\frac{1}{\|\epsilon v\|_{L_\psi^1}}\int_\mathbb{R}\epsilon \psi(x)v(x)dx=1,\]

\[F_{uu}(v)\equiv \lim_{\epsilon\rightarrow0}\frac{1}{\|\epsilon v\|_{L_\psi^1}}(F_u(u+\epsilon v)-F_u(u))=0\]

for each \(v\in L_\psi^2\). In this case \(X(t)\) is a cylindrical Wiener process and is hence \(L_\psi^2\) valued. Also, since \(F\) is mapping objects from \(L_\psi^2\) to \(\mathbb{R}\), we know the inncer product in Itô’s formula is with respect to \(L_\psi^2\). Therefore, applying Itô’s formula, we get

\[F(B)=F(B(0))+\int_0^t\left<F_u(B(s)),dB(s)\right>_{L_\psi^2}+\frac{1}{2}\mathrm{Tr}(F_{uu}(B(s))I)ds\]

\[=\int_0^t\left<1,dB(s)\right>_{L_\psi^2}=\int_0^t\int_\mathbb{R}\psi(x)dB(s,x)dx=\int_\mathbb{R}\psi(x)\int_0^tdB(s,x)dx\]

\[=\int_\mathbb{R}\psi(x)B(t,x)dx\]

as desired.

  • BW: (returning to the original problem) Now suppose we are working with

\[\mathfrak{n}(t,z)=m(z)\mathfrak{n}(t,z)dt+\eta\mathfrak{n}(t,z)dB(t,z).\]

In this case \(\mathfrak{n}(0)=X(0)\), \(\mathfrak{n}(t)=X(t)\) and \(\phi(t)=m\mathfrak{n}(t)\) are \(L^2(\mathbb{R})\)-valued and \(\Phi(t):L^2_{\mathfrak{n}(t)}(\mathbb{R})\rightarrow L^2(\mathbb{R})\) defined by \(B\mapsto\eta\mathfrak{n}(t)B\) for each \(t\) converts the cylindrical Wiener process \(B(t,z)\) into a \(L^2(\mathbb{R})\)-valued random variable. We define \(F:L^2(\mathbb{R})\rightarrow\mathbb{R}\) by \(F(v)=\int_\mathbb{R}|v(x)|dx\) so that \(F_t=0\) and

\[F_u(v)=\lim_{\epsilon\rightarrow0}\frac{1}{\|\epsilon v\|_1}(F(u+\epsilon v)-F(u))\]

\[=\lim_{\epsilon\rightarrow0}\frac{1}{\|\epsilon v\|_1}\int_\mathbb{R}|\epsilon v(x)|dx=1,\]

\[F_{uu}(v)=0.\]

Then, applying Itô’s formula,

\[F(\mathfrak{n}(t))=F(\mathfrak{n}(0))+\int_0^t\left<1,\eta\mathfrak{n}(t) dB(s)\right>_{L^2}+\int_0^t\left<1,m(z)\mathfrak{n}(s)\right>ds\]

\[=n(0)+\eta\int_0^tn(s)\int_\mathbb{R}p(s,z)dB(s,z)dz+\int_0^tn(t)\int_\mathbb{R}m(z)p(s,z)dzds\]

Thus, the real valued process \(n(t)\) satisfies the sde

\[dn(t)=n(t)\tilde{m}(t)dt+n(t)\tilde\eta(t)dW(t)\]

where

\[\tilde m(t)=\int_\mathbb{R}m(z)p(t,z)dz,\]

\[\tilde\eta(t)dW(t)=\eta\int_\mathbb{R}p(t,z)dB(t,z)dz\]

and \(W\) is a standard one dimensional Brownian motion. Since the Malthusian fitness of the whole population \(\tilde m\) is a function of time, \(n(t)\) is not a geometric Brownian motion as intuition might suggest.

A.C Nuclear & Hilbert-Schmidt operators

Let \(L(E,G)\) be the Banach space of all linear bounded operators from \(E\) into \(G\) with the supremum norm

\[\|T\|\equiv\sup_{x\in E}\frac{\|Tx\|}{\|x\|}.\]

  • Def: \(T\in L(E,G)\) is nuclear or trace class if there exists sequences \(\{a_j\}\subset G, \ \{\phi_j\}\subset E^*\) s.t.

\[\sum_{j=1}^\infty\|a_j\|\|\phi_j\|<\infty\]

and \(T\) has the representation \(Tx=\sum_ja_j\phi_j(x), \ x\in E\).

The space of all nuclear operators with the norm

\[\|T\|_1\equiv\inf\left\{\sum_j\|a_j\|\|\phi_j\|:Tx=\sum_ja_j\phi_j(x)\right\}\]

is a Banach space denoted as \(L_1(E,G)\) or \(L_1(E)\) if \(E=E\). Let \(K\) be another Banach space. If \(T\in L_1(E,G)\) and \(S\in L_1(G,K)\), then \(TS\in L_1(E,K)\) and \(\|TS\|_1\leq\|T\|_1\|S\|_1\). If \(T\in L_1(U)\) and \(\{e_k\}\) is a complete orhonormal system in \(U\), then we define

\[\mathrm{Tr}T\equiv\sum_j\left<Te_j,e_j\right>\]

  • Proposition C.3: A nonnegative operator \(T\in L(H)\) is of trace class iff there exists an orthonormal basis \(\{e_k\}\) on \(H\) s.t.

\[\sum_j\left<Te_j,e_j\right><\infty.\]

When this occurs \(\mathrm{Tr}T=\|T\|_1\).

  • Def: For two separable Hilbert spaces \(U_1\) and \(U_2\) with complete orthonormal bases \(\{e_k\}\subset U_1\) and \(\{f_k\}\subset U_2\), a linear bounded operator \(T:U_1\rightarrow U_2\) is said to be Hilbert-Schmidt if

\[\sum_k|Te_k|^2<\infty.\]

The space of all Hilbert-Schmidt operators from \(U_1\) to \(U_2\), equipped with the norm

\[\|T\|_2\equiv\left(\sum_k|Te_k|^2\right)^{1/2}\]

is a separable Hilbert space with the scalar product

\[\left<S,T\right>\equiv\sum_k\left<Se_k,Te_k\right>.\]

  • Notation: (outer product) For any vectors \(a\in U_1\) and \(b\in U_2\), the operation \(b\otimes a\) denotes the linear operator defined by

\[b\otimes a:U_1\rightarrow U_2, \ h\mapsto \left<a,h\right>b, \ \ h\in U_1.\]

The double sequence \(\left\{f_j\otimes e_k\right\}_{j,k\in \mathbb{N}}\) is a complete orthonormal basis in \(L_2(E,F)\).

  • Proposition C.4: Let \(U_1,U_2,U_3\) be separable Hilbert spaces. If \(T\in L_2(U_1,U_2)\) and \(S\in L_2(U_2,U_3)\) then \(ST\in L_1(U_1,U_3)\) and

\[\|ST\|_1\leq\|S\|_2\|T\|_2.\]
