Harmonic Analysis 02-25-19

Review:

  • \(e^{-\pi x^2}\mapsto e^{-\pi\gamma^2}\)

  • \(K_\delta(x)\equiv\frac{1}{\sqrt\delta}e^{-\pi x^2/\delta}\mapsto e^{-\pi\delta\gamma^2}, \ \delta>0\)

  • \(\{K_\delta\}\) is a family of approximate identities (or good kernels). That is:
    • \(\int_\mathbb{R}K_\delta=1\)
    • \(\int_\mathbb{R}|K_\delta|\leq M\)
    • \(\int_{|x|>\eta}|K_\delta|\rightarrow 0, \ \delta\rightarrow 0\)

  • If \(f\in S(\mathbb{R})\), then \((f*K_\delta)\rightarrow f\) as \(\delta\rightarrow 0\), uniformly.

Proposition: (Multiplication formula)

If \(f,g\in S(\mathbb{R})\) then

\[\int_\mathbb{R}f\hat g=\int_\mathbb{R}\hat f g.\]

Proof: First we state some analogue of Fubini’s theorem. Suppose \(F(x,y)\) is continuous in \(\mathbb{R}^2\). Assume

\[|F(x,y)|\leq\frac{A}{(1+x^2)(1+y^2)}.\]

  • The set \(M(\mathbb{R})\) consists of functions \(F(x)\) s.t. \(|F(x)|\leq\frac{A}{1+x^2}\) and is called the set of functions of moderate decrease.

Note that \(F(x,y)\in M(\mathbb{R})\) when either \(x\) or \(y\) are fixed. Further more

\[F_1(x)\equiv\int_\mathbb{R}F(x,y)dy,F_2(y)\equiv\int_\mathbb{R}F(x,y)dx\in M(\mathbb{R})\]

and are continuous. Finally,

\[(\star) \ \int_\mathbb{R}F_1=\int_\mathbb{R}F_2=\int_{\mathbb{R}^2}F\]

which implies that the order of integration can be freely interchanged. Now set

\[F(x,y)=f(x)g(x)e^{-2\pi i xy}\]

Then \(F_1=f\hat g\) and \(F_2=\hat f g\). Hence, by \((\star)\), the proof is complete. \(\square\)

Theorem: (Fourier Inversion)

If \(f\in S(\mathbb{R})\) then \(f(x)=\int_\mathbb{R}\hat f(\gamma)e^{2\pi i x\gamma}dx\).

Proof:

Let \(G_\delta(x)=e^{-\pi\delta x^2}\). Then \(\hat G_\delta(\gamma)=\frac{1}{\sqrt\delta}e^{-\pi\gamma^2/\delta}=K_\delta(\gamma)\). By the multiplcation formula

\[\int_\mathbb{R}f(x)\hat G_\delta(x)dx=\int_\mathbb{R}\hat f(\gamma) G_\delta(\gamma)d\gamma\]

or

\[\int_\mathbb{R}f(x)K_\delta(x)dx=\int_\mathbb{R}\hat f(\gamma) G_\delta(\gamma)d\gamma\]

\[(f*K_\delta)(0)=\int_\mathbb{R}\hat f(\gamma)G_\delta(\gamma)d\gamma\]

Recall: \(\{K_\delta\}\) is a good kernel. So, as \(\delta\rightarrow 0\)

\[f(0)=\int_\mathbb{R}\hat f(\gamma)d\gamma\]

(requires interchange of limit and integral, justify by Lebesgue’s dom conv).

In general, \(F(y)\equiv f(y+x)\). Then

\[f(x)=F(0)=\int_\mathbb{R}\hat F(\gamma)d\gamma=\int_\mathbb{R}\hat f(\gamma)e^{2\pi i x\gamma}d\gamma. \ \square\]

Define:

  • \(\mathscr{F}:S(\mathbb{R})\rightarrow S(\mathbb{R}), \ f(x)\mapsto\hat f(\gamma)=\int_\mathbb{R}f(x)e^{-2\pi i x\gamma}dx\)
  • \(\mathscr{F}^*:S(\mathbb{R})\rightarrow S(\mathbb{R}), \ \hat f(\gamma)\mapsto f(x)=\int_\mathbb{R}\hat f(\gamma)e^{2\pi i x\gamma}dx\)

By the inversion formula \(\mathscr{F}^*\circ\mathscr{F}=\mathscr{F}\circ\mathscr{F}^*=\mathrm{Id}\) and, hence, \(\mathscr{F}\) is a bijection on \(S(\mathbb{R})\).
