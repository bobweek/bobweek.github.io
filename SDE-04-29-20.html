The Goal

  • How do we make sense of \(\left\{\begin{matrix} dX=b(X,t)dt+B(X,t)dW \\ X(0)=X_0 \end{matrix}\right.\) ?

  • Rewrite as \(X(t)=X_0+\int_0^tb(X,s)ds+\int_0^tB(X,s)dW(s)\).

  • Can make sense of \(\int_0^tb(X,s)ds\) for each fixed \(\omega\).

    • That is, for each sample path, sometimes referred to as an “\(\omega\)-by-\(\omega\)” argument.

  • But what is \(\int_0^tB(X,s)dW(s)\) ?

    • Today’s topic!!

A first definition

The Paley-Wiener-Zygmund (PWZ) stochastic integral

  • Denote the set of continuously differentiabl \(g:[0,T]\to\mathbb{R}\) with \(g(0)=g(T)=0\) by \(C^1(0,T)\)

  • For \(g\in C^1(0,T)\) define

  • \[\int_0^Tg(t)dW(t):=-\int_0^Tg'(t)W(t)dt.\]

  • If we pretend \(W(t)\) is continuously differentiable, this coincides with integration by parts \(\int_a^b udv=[uv]_a^b-\int_a^b vdu\).

Properties of the PWZ integral

  1. \(\mathbb{E}\left(\int_0^TgdW\right)=0\)

  2. \(\mathbb{E}\left[\left(\int_0^TgdW\right)^2\right]=\int_0^T g^2(t)dt\)

  • We assumed \(g\in C^1(0,T)\). Since \([0,T]\) is a finite interval we have \(g\in L^2(0,T)\) where …

  • \(L^2(0,T)\) is the space of functions \(f\) on \([0,T]\) (not necessarily differentiable) such that \(\int_0^Tf^2(t)dt<+\infty\).

  • Hence, \(C^1(0,T)\subset L^2(0,T).\)

The “standard machine”

A general method to extend operators

  • Observe properties that characterize an operator defined on a set of “nice” functions.

  • Try to define the operator on a larger set of functions while preserving the characteristic properties.

  • The larger set of functions must contain the subset of “nice” functions in a special way.

    • We need the larger set of functions to be “complete” so that limits converge.

    • We need to approximate the general functions arbitrarily well with “nice” functions.

What is completeness?

  • Let \((S,d)\) be a metric space so that for any \(x,y\in S\), \(d(x,y)\) is the “distance” between \(x\) and \(y\).

  • A Cauchy sequence in \((S,d)\) is a sequence of points \(\{x_1,x_2,\dots\}\) such that \(d(x_n,x_m)\to0\) as \(n,m\to\infty\).

  • Some spaces allow Cauchy sequences that fail to converge.

  • Metric spaces in which all Cauchy sequences converge are called complete.

  • For any metric space \(S\), it’s completion \(\bar S\) the union of \(S\) and limits of all Cauchy sequences (equivalent limits are identified).

  • If \(U\subset S\), \(S\) a complete metric space, then \(U\) is dense in \(S\) if \(\bar U=S\).

An example

  • Consider \(S=(0,1]\) with \(d(x,y)=|x-y|\).

  • The sequence given by \(x_n=\frac{1}{n}\) satisfies \(d(x_n,x_m)\to0\) as \(n,m\to\infty\) and hence is Cauchy.

  • But \(x_n\to0\notin S\).

  • If we enlarge \((0,1]\) to \(\overline{(0,1]}=[0,1]\) then all Cauchy sequences converge.

  • Hence \([0,1]\) is complete with respect to the metric \(d(x,y)=|x-y|\) and \((0,1]\) is dense in \([0,1]\).

A more useful example

  • For the set \(S\) we define \(L^2(S)\) as the Lebesgue space of square-integrable functions on \(S\).

  • That is, \(f\in L^2(S)\) if \(\int_Sf^2(s)ds<+\infty\).

  • We will see \(L^2\) spaces often, so it is useful to note they are complete.

PWZ as an operator on \(C^1(0,T)\)

  • Can think of PWZ integral as an operator mapping functions \(g\in C^1(0,T)\) to random variables.

  • Denote PWZ by \(\mathcal{I}:C^1(0,T)\to L^2(\Omega)\)

    • \(L^2(\Omega)\) is the set of random variables with finite variance.

    • For \(X,Y\in L^2(\Omega)\) we can measure their distance with \(\mathbb{E}\left[\left(X-Y\right)^2\right]\).

  • To summarize, for any \(g\in C^1(0,T)\): \[\mathbb{E}\left(\mathcal{I}(g)\right)=0\] \[\mathbb{V}\left(\mathcal{I}(g)\right)=\mathbb{E}\left[\left(\mathcal{I}(g)\right)^2\right]=\int_0^Tg^2(t)dt.\]

\(C^1(0,T)\) and \(L^2(0,T)\)

  • For \(f,g\in L^2(0,T)\) we measure the “distance” between them with \[d(f,g):=\int_0^t\big(f(t)-g(t)\big)^2dt.\]

  • \(C^1(0,T)\) is dense in \(L^2(0,T)\) (requires proof).

  • Thus, for any \(f\in L^2(0,T)\), we can find \(\{g_1,g_2,\dots\}\subset C^1(0,T)\) such that \[\lim_{n\to\infty}d(f,g_n)=0.\]

An example

Extending PWZ

  • Have: \(\mathcal{I}(g_n)\in L^2(\Omega)\) by definition.

  • Want: \(\lim_{n\to\infty}\mathcal{I}(g_n)\in L^2(\Omega)\).

  • Property 2 of PWZ implies the isometry (same distance):

  • \[\mathbb{E}\left[\left(\mathcal{I}(g_m)-\mathcal{I}(g_n)\right)^2\right]=d(g_m,g_n).\]

  • Hence, the sequence \(\{\mathcal{I}(g_n)\}_{n=1}^\infty\) converges in \(L^2(\Omega)\).

  • We can therefore define \(\mathcal{I}(f):=\lim_{n\to\infty}\mathcal{I}(g_n)\).

  • That is, \(\int_0^Tf(t)dW(t):=\lim_{n\to\infty}\int_0^T g_n(t)dW(t)\).

Before you break out the champagne

  • PWZ only works for deterministic functions, not stochastic processes.

  • How should we define \(\int_0^T W(t)dW(t)\)?

  • Evans suggests we try a Reimann sum.

Reimann sum approximation of

\(\int_0^T W(t)dW(t)\)

  1. \(P:=\{0=t_0<t_1<\cdots<t_m=T\}\) is a partition of \([0,T]\) with mesh size \(|P|:=\max_k|t_{k+1}-t_k|\).

  2. For \(\lambda\in[0,1]\) set \(\tau_k:=(1-\lambda)t_k+\lambda t_{k+1}\) so that \(\tau_k\in[t_k,t_{k+1}]\).

  3. The Reimann sum approximation is \[R(P,\lambda)=\sum_{k=0}^{m-1}W(\tau_k)\big(W(t_{k+1})-W(t_k)\big)\]

  • What happens as \(|P|\to0\) for fixed \(\lambda\)? Before we answer this, we introduce…

Quadratic variation (QV)

  • Let \([a,b]\subset[0,+\infty)\) and set \(P^n:=\{a=t_0^n<\cdots<t_{m_n}^n=b\}\) such that \(\lim_{n\to\infty}|P^n|=0\).

  • Then, \(Q_n:=\sum_{k=1}^{m_n-1}\Big(W(t_{k+1}^n)-W(t_k^n)\Big)^2\to (b-a)\) in \(L^2(\Omega)\)

  • That is, \(\lim_{n\to\infty}\mathbb{E}\left[\big(Q_n-(b-a)\big)^2\right]=0\).

Scaling relationship curiousity

  • Evans states the result on QV partially explains \((dW)^2\approx dt\).

  • To illustrate, recall the Reimann integral is given by \[\sum_{k=0}^{m_n-1}t^n_{k+1}-t^n_k\to\int_a^bdt=b-a.\]

  • Then the QV result suggests \(\int_0^T(dW)^2=\int_0^Tdt\).

Returning to the question:

What happens to \(R(P,\lambda)\) when \(|P|\to0\)?

  • Define \(R_n:=R(P^n,\lambda)=\sum_{k=0}^{m_n-1}W(\tau_k^n)\Big(W(t_{k+1}^n)-W(t_k^n)\Big)\).
  • Then \(R_n=\frac{W(T)^2}{2}-A+B+C\) where:
    • QV \(\implies A_n=\frac{1}{2}\sum_{k=1}^{m_n-1}\Big(W(t_{k+1}^n)-W(t_k^n)\Big)^2\to\frac{T}{2}\)
    • QV \(\implies B_n=\sum_{k=1}^{m_n-1}\Big(W(\tau_k^n)-W(t_k^n)\Big)^2\to\lambda T\)
    • Independent increments \(\implies C_n=\sum_{k=1}^{m_n-1}\Big(W(t_{k+1}^n)-W(\tau_k^n)\Big)\Big(W(\tau_k^n)-W(t_k^n)\Big)\to0\)

Hence!

\[\lim_{n\to\infty}R_n=\lim_{n\to\infty}\sum_{k=0}^{m_n-1}W(\tau_k^n)\Big(W(t_{k+1}^n)-W(t_k^n)\Big)=\frac{W(T)^2}{2}+\left(\lambda-\frac{1}{2}\right)T\]

  • Setting \(\lambda=0\) coincides with the Itô integral \[\int_0^T WdW=\frac{1}{2}\Big(W(T)^2-T\Big).\]

  • Setting \(\lambda=\frac{1}{2}\) coincides with the Stratonovich integral \[\int_0^TW\circ dW=\frac{W(T)^2}{2}.\]

Why is \(\lambda=0\) a sensible choice?

Part one

  • Consider the Reimann sum for integrating a process \(G(t)\) against \(W(t)\): \[\sum_{k=0}^{m-1}G(\tau_k)\Big(W(t_{k+1})-W(t_k)\Big)\]

  • If we choose \(\lambda>0\) then \(\tau_k>t_k\) requires us to “look ahead” at a future state of \(G(t)\).
    • Processes that “look ahead” are formalized as a predictable or anticipating processes.
    • Processes that do not “look ahead” are formalized as adapted or nonanticipating processes.

Why is \(\lambda=0\) a sensible choice?

Part two

  • Consider the SDE \(dX=a(X,t)dt+B(X,t)dW\) and ODE \(\frac{dx}{dt}=a(x,t)\).

  • Given solutions to both exist, we might expect \(\mathbb{E}(X(t))=x(t)\).

  • This only holds for \(\lambda=0\).

  • Hence this desired property requires us to work with adapted processes.

  • The next few slides introduce notions that formalize adapted processes.

\(\sigma\)-algebra review

  • A \(\sigma\)-algebra \(\mathcal{S}\) on a set \(S\) is a collection of subsets of \(S\) (referred to as \(\mathcal{S}\)-measurable sets) that satisfies:

    • \(\emptyset,S\in\mathcal{S}\)
    • \(U\in \mathcal{S}\implies S\setminus U\in \mathcal{S}\)
    • \(\{U_n\}_{n=1}^\infty\subset\mathcal{S}\implies \cap_nU_n,\cup_nU_n\in\mathcal{S}\).

  • An important example is the Borel \(\sigma\)-algebra on \(\mathbb{R}\), denoted \(\mathcal{B}(\mathbb{R})\).

  • It is generated by taking unions, intersections and complements of intervals.

  • Elements of \(\mathcal{B}(\mathbb{R})\) are the “typical” sets we tend to think of.

  • By focusing on \(\mathcal{B}(\mathbb{R})\) we avoid issues that arise from pathological sets that are hard to imagine, but can be proved to exist by the Axiom of Choice.

Random variable as measurable functions

  • For \(X:\Omega\to\mathbb{R}\) and a \(\sigma\)-algebra \(\mathcal{U}\), we say \(X\) is \(\mathcal{U}\)-measurable and write \(X\in\mathcal{U}\) if:
    • for all \(B\in\mathcal{B}(\mathbb{R})\), we have

  • \[X^{-1}(B)=\{\omega\in\Omega \ | \ X(\omega)\in B \}\in\mathcal{U}.\]

  • That is, \(X\) is measurable if the pre-image of unions, intersections and complements of intervals can be found in \(\mathcal{U}.\)

  • The collection \(\sigma(X)=\{X^{-1}(B) \ | \ B\in\mathcal{B}(\mathbb{R})\}\) is a sub-\(\sigma\)-algrebra of \(\mathcal{U}\).

  • We call \(\sigma(X)\) the \(\sigma\)-algebra generated by \(X\).

Filtrations and adapted processes

  • Fix the \(\sigma\)-algebra \(\mathcal{U}\) and suppose
    • for each \(t\geq0\) that \(\mathscr{F}(t)\) is a sub-\(\sigma\)-algebra of \(\mathcal{U}\)
    • the collection \(\{\mathscr{F}\}:=\{\mathscr{F}(t)\}_{t\geq0}\) satisfies \(\mathscr{F}(s)\subset\mathscr{F}(t)\) when \(s\leq t\).

  • Then \(\{\mathscr{F}\}\) is called a filtration of \(\mathcal{U}\).

  • Can think of \(\mathscr{F}(t)\) as the set of all possible events up to time \(t\).
    • Or, as the information available for a process up to time \(t\).

  • If \(X(t)\in\mathscr{F}(t)\) for each \(t\geq0\) the process \(X(\cdot)\) is adapted to \(\{\mathscr{F}\}\).

  • In particular, \(\{\sigma(X(t))\}_{t\geq0}\) is a filtration and \(X(\cdot)\) is adapted to it.

Why is this a sensible definition of an adapted process?

Here’s my attempt to answer this.

  • Suppose \(\{\mathscr{F}\}\) is a filtration and \(X(t)\notin\mathscr{F}(t)\).

  • Then there is an event \(A\) describing an observable state of \(X(t)\) such that \(A\notin\mathscr{F}(t)\).
    • That is, \(\mathscr{F}(t)\) does not allow \(A\) to occur at time \(t\).

  • But \(\mathscr{F}(s)\subset\mathscr{F}(t)\) for \(s\leq t\) and so \(A\notin\mathscr{F}(s)\) for \(s\leq t\).

  • Presumably then there exists some future time \(r>t\) such that \(A\in\mathscr{F}(r)\).

  • But \(\mathscr{F}(r)\), and thus \(X(t)\), contain information about the future. Hence, \(X(t)\) is called predictable, or, anticipating.

Adapted processes are not good enough

  • A stochastic process \(X(t)=X(t,\omega)\) is a bivariate function on \([0,\infty)\times\Omega\).
    • Sample space: \((\Omega,\mathcal{U},\mathbb{P})\)

  • Hence \(\mathbb{E}\left(\int_0^TX(t)dt\right)=\int_\Omega\int_0^TX(t,\omega)dt\mathbb{P}(d\omega)\) is an integral on the product space \([0,\infty)\times\Omega\).

  • Need \(X(t,\omega)\) to be measurable wrt the product \(\sigma\)-algebra \(\mathcal{B}([0,\infty))\otimes\mathcal{U}\)

    • generated by unions, intersections and complements of the sets \(A\times B\) for \(A\in\mathcal{B}([0,\infty))\) and \(B\in\mathcal{U}\).

  • A concept that combines these notions is called a progressively measurable process.

Progressively measurable processes

  • Suppose \((\Omega,\mathcal{U},\mathbb{P})\) is a probability space and

    • \(\{\mathscr{F}\}\) a filtration of \(\mathcal{U}\),

    • \(X:[0,\infty)\times\Omega\to\mathbb{R}\) a real-valued stochastic process,

  • \(X\) is progressively measurable wrt \(\{\mathscr{F}\}\) if, for each \(T\geq0\), its restriction to \([0,T]\times\Omega\) is

  • \[\mathcal{B}([0,T])\otimes\mathscr{F}(T)\text{-measurable.}\]

Progressively measurable (PM) processes

Summary

  • The technical definition of a PM process implies it is an adapted process.

  • This means PM processes do not depend on information about the future.

  • PM processes are also defined so we can integrate over both time and \(\Omega\).

    • This is important for taking expectations of integrals.

Integrable and square-integrable PM processes

  • If \(G\) is PM and \(\mathbb{E}\left(\int_0^T|G(t)|dt\right)<+\infty\), then we write \[G\in\mathbb{L}^1(0,T).\]

    • These processes will be important for next week’s discussion of Itô’s chain rule.

  • If \(G\) is PM and \(\mathbb{E}\left(\int_0^TG^2(t)dt\right)<+\infty\), then we write \[G\in\mathbb{L}^2(0,T).\]

    • We will define Itô’s integral as an operator on \(\mathbb{L}^2(0,T)\).

Step processes

Approximations of \(\mathbb{L}^2(0,T)\) processes

  • If there is a partition \({\{t_k}\}_{k=1}^{m}\) of \([0,T]\) and a set of random variables \(\{G_0,\dots,G_{m-1}\}\) such that

    • \(G\in\mathbb{L}^2(0,T)\) satisfies \[G(t)=G_k, \ t\in[t_k,t_{k+1}), \ k=0,\dots,m-1,\]

    • then \(G\) is called a step process.

  • The set of step processes is a “nice” subspace of \(\mathbb{L}^2(0,T)\).

  • In particular, we can approximate \(\mathbb{L}^2(0,T)\) processes arbitrarily close with step processes.

Approximating \(\mathbb{L}^2(0,T)\) processes

Itô’s integral

For step processes

  • For step processes \(G\) we define the Itô integral by \[\mathscr{I}(G):=\int_0^TGdW:=\sum_{k=0}^{m-1}G_k\big(W(t_{k+1})-W(t_k)\big).\]

Properties of Itô’s integral

For step processes

  • Let \(G,H\) be step processes. Then \(G,H\) are PM and we can thus prove

    1. \(\int_0^TaG+bHdW=a\int_0^TGdW+b\int_0^THdW\dots\) linearity,

    2. \(\mathbb{E}\left(\int_0^TGdW\right)=0\dots\) martingale property,

    3. \(\mathbb{E}\left[\left(\int_0^TGdW\right)^2\right]=\mathbb{E}\left(\int_0^TG^2dW\right)\dots\) isometry.

Return of the standard machine

  • Using properties of Lebesgue integration, we can show \(\mathbb{L}^2(0,T)\) is complete.

  • In particular, Cauchy sequences consisting of step processes converge in \(\mathbb{L}^2(0,T)\).

  • However, the standard machine requires us to show the space of step processes is dense in \(\mathbb{L}^2(0,T)\).

  • That is, we need for any \(G\in\mathbb{L}^2(0,T)\) a sequence of step processes \(G_n\) such that \[\mathbb{E}\left(\int_0^T(G-G_n)^2dt\right)\to0.\]

Step processes from \(\mathbb{L}^2(0,T)\) processes

  • For \(G\in\mathbb{L}^2(0,T)\), set \[G_m(t):=\int_0^tme^{m(s-t)}G(s)ds.\]

  • Then \(G_m\) is a continuous approximation of \(G\) such that \(d(G_m,G)\to0\) a.s. as \(m\to\infty\).

  • Set \(G_{n,m}(t):=G_m(k/n)\) for \(t\in[k/n,(k+1)/n)\) and \(k=0,\dots,[nT]\).

  • Then \(G_{n,m}\) is a step function and \(d(G_{n,m},G)\to0\) a.s. as \(n,m\to\infty\).

Itô’s integral

For \(\mathbb{L}^2(0,T)\) processes

  • We can find a Cauchy sequence \(\{G_n\}\) of step processes that converge to \(G\) in \(\mathbb{L}^2(0,T)\).

  • The isometry property of Itô’s integral for step processes implies

  • \[\mathbb{E}\left[\left(\int_0^TG_n-G_mdW\right)^2\right]=\mathbb{E}\left(\int_0^T(G_n-G_m)^2dt\right)\to0.\]

  • Since \(L^2(\Omega)\) is complete, limit \(\mathscr{I}(G):=\lim_{n\to\infty}\int_0^TG_ndW\) exists!

Properties of Itô’s integral

  • In analogy with step processes, we find for all \(G,H\in\mathbb{L}^2(0,T)\)

    1. \(\mathscr{I}(aG+bH)=a\mathscr{I}(G)+b\mathscr{I}(H)\),

    2. \(\mathbb{E}\left[\mathscr{I}(G)\right]=0\),

    3. \(\mathbb{E}\left[\mathscr{I}(G)^2\right]=\mathbb{E}\left(\int_0^TG^2dt\right).\)

  • However, we now have the additional property

    1. \(\mathbb{E}\left[\mathscr{I}(G)\mathscr{I}(H)\right]=\mathbb{E}\left(\int_0^TGHdt\right)\dots\) orthogonality.
