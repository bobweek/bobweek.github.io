Harmonic Analysis 02-06-2019

Completeness of the trigonometric system \(\{e^{inx}\}\)

\[L^2[-\pi,\pi)=\left\{f:\int_{-\pi}^\pi|f(x)|^2dx<\infty\right\}\]

\[\mathbb{S}^1\sim[-\pi,\pi)\]

\[\left<f,g\right>=\frac{1}{2\pi}\int_{-\pi}^\pi f(x)\bar g(x)dx\]

\[\|f\|^2=\left<f,f\right>\]

  • \(f,g\) are orthogonal if \(\left<f,g\right>=0\), write as \(f\perp g\).

  • if two sets \(S_1,S_2\) are orthogonal then we write \(S_1\perp S_2\) means that for any \(f\in S_1\) and \(g\in S_2\) we have \(f\perp g\).

  • we know \(\{e_n=e^{inx}\}_{n\in\mathbb{Z}}\) is orthonormal

Let \(\{f_n\}_{n=1}^\infty\) be an orthonormal family in \(L^2[-\pi,\pi)\). The family \(\{f_n\}\) is said to be complete, or a complete orthonormal system, or an orthonormal basis if for each \(f\in L^2\) there exists \(\{a_n\}_{n=1}^\infty\subset\mathbb{R}\) s.t. \[\|f-\sum_{n=1}^N a_n f_n\|\rightarrow 0\] as \(N\rightarrow\infty\).

Equivalent conditions for completeness

If \(X=\{f_n\}\) is an orthonormal family in \(L^2\), then the following are equivalent:

  1. \(X\) is complete (or an ONB)

  2. if \(f\in L^2\) and \(f\perp X\), then \(f=0\)

  3. Plancherel’s Identity: For each \(f\in L^2\) \[\|f\|^2=\sum_{n=1}^\infty|\left<f,f_n\right>|^2\] (In the special case when \(f_n=e_n\), this identiy is called Parseval’s Identity)

Lemma Uniqueness of coefficients

If \(\{f_n\}\) is an ONB in \(L^2\), then \(f\) can be written as \(f=\sum a_nf_n\) where the coeff’s are given by

\[a_n=\left<f,f_n\right>.\]

Proof: Let \(f=\sum a_n f_n\). Fix \(k\in\mathbb{N}\). Take inner product on both sides with \(f_k\): \[\left<f,f_k\right>=\left<\sum a_nf_n,f_k\right>\] by Fubini \[a_k=\left<f,f_k\right>=\sum\left<f_n,f_k\right>=a_k. \ \square\]

Stuff

Take \(\{f_n\}=\{e_n\}\) so that \(S_N=\sum_{|n|\leq N}\hat f(n)e^{inx}\) and

\[\hat f(n)=\frac{1}{2\pi}\int_{-\pi}^\pi f(x)e^{-inx}dx=\left<f,e_n\right>.\]

From the prev lectures:

  • \(\|f-S_Nf\|\rightarrow 0\) as \(N\rightarrow\infty\). Thus \(\{e_n\}\) is an ONB in \(L^2\).

\[(\star) \ \|f\|^2=\|f-S_Nf\|^2+\sum_{|n|\leq N}|\hat f(n)|^2=\|f-S_Nf\|^2+\|S_Nf\|^2\]

This implies Bessel’s ineq: \(\|f\|^2\geq\|S_Nf\|^2\)

Take \(N\rightarrow\infty\) on each side of \((\star)\):

\[\|f\|^2=0+\sum_{n\in\mathbb{Z}}|\left<f,e_n\right>|^2\]

which is Parseval’s. All of these hold for any ONB (Plancherel’s).

Another way to establish Plancherel’s:

Since \(\{f_n\}\) is an ONB

\[f=\sum\left<f,f_n\right>f_n\implies\left<f,f\right>=\left<\sum\left<f,f_n\right>,f\right>\]

\[\|f\|^2=\sum\left<f,f_n\right>\left<f_n,f\right>\]

\[\|f\|^2=\sum\left<f,f_n\right>\overline{\left<f,f_n\right>}\]

\[\|f\|^2=\sum|\left<f,f_n\right>|^2\]
