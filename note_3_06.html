Harmonic Analysis 03-06-19

  • Recap:

\[v=(z_0,\dots,z_{N-1})\in\mathbb{C}^N\]

The discrete Fourier transform (DFT) of \(v\) is \(\hat v\in\mathbb{C}^N\) is defined by

\[\hat v(n)\equiv\frac{1}{\sqrt N}\sum_{k=0}^{N-1}z_ke^{-2\pi ikn/N}\]

Define the vectors \(\{e_n\}_{n=0}^{N-1}\subset\mathbb{C}^N\) by

\[e_n\equiv\frac{1}{\sqrt{N}}e^{2\pi ikn/N}.\]

Then \(\{e_n\}\) is an ONB of \(\mathbb{C}^N\). Hence

\[\sum_{n=0}^{N-1}\left<v,e_n\right>e_n(k)=\frac{1}{\sqrt N}\sum_{n=0}^{N-1}\left<v,e_n\right>e^{2\pi iknN}\]

and \(\hat v(n)=\left<v,e_n\right>\).

Parseval’s/Plancherel’s Identity:

If \(v\in\mathbb{C}^N\) then

\[v(k)=\sum_n\hat v(n)e_n(k)\]

and

\[\sum_n|\hat v(n)|^2=\sum_n|v(n)|^2\]

ie, \(\|\hat v\|=\|v\|\).

  • Proof:

Since we already noted \(\{e_n\}\) is an ONB, the first result is already demonstrated. For the second part note:

\[|v(k)|^2=v(k)\overline{v(k)}\]

\[=\sum_n\left<v,e_n\right>e_n(k)\sum_m\overline{\left<v,e_m\right>e_m(k)}\]

\[=\sqrt{N}\sum_n\sum_m\left<v,e_n\right>\overline{\left<v,e_m\right>}e_{n-m}\]

\[=\sqrt{N}\left(\sum_{n=m}\left<v,e_n\right>\overline{\left<v,e_n\right>}+\sum_{n\neq m}\left<v,e_n\right>\overline{\left<v,e_m\right>}e_{n-m}\right)\]

\[=\sqrt{N}\left(\sum_{n=m}|\left<v,e_n\right>|^2+\sum_{n\neq m}\left<v,e_n\right>\overline{\left<v,e_m\right>}e_{n-m}\right)\]

Then

\[\sum_k|v(k)|^2=\sum_n|\left<v,e_n\right>|^2=\sum_n|\hat v(n)|^2.\]

DFT matrix

We have \(v=\sum_n\hat v(n)e_n\) and hence can be written in matrix form as

\[v=\left(\begin{matrix} \vdots&\vdots&&\vdots\\ e_0&e_1&\cdots&e_{N-1}\\ \vdots&\vdots&&\vdots\end{matrix}\right)\hat v\]

\[=F_N\hat v.\]

The matrix \(F_N\) is referred to as the Fourier matrix (or DFT matrix). Here are some nice properties:

  • \(\hat v=\bar{F_N}^\top v={F_N}^* v\), where \(^*\) denotes Hermetian (adjoint)

Note that \(F_N\) is always invertible since its columns are lin-ind by def. Furthermore, since \(\hat v={F_N}^{-1} v\), we have

  • \({F_N}^{-1}={F_N}^*\implies\) \(F_N\) is unitary (hence isometry)

  • \(F_N\) is symmetric
